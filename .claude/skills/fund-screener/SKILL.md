# 公募基金筛选器

> **必须先读取 `FUND_ANALYSIS_WORKFLOW.md`，严格按其中 4 步流程执行。**

根据用户设定的多维度条件，从全市场公募基金中筛选出符合条件的基金池，并打分排序。

## 触发条件

- "筛选基金" / "推荐基金" / "什么基金好" / "帮我选"
- "买什么基金" / "怎么买基金" / "XX 万元买什么"
- "XX 基金怎么样" / "帮我看看这只基金"

## 工作流

### 第一步：确认筛选参数

向用户询问以下参数（括号内为默认值）：

1. **方向**：股票型 / 混合型 / 债券型 / 指数型 / QDII / 全部
2. **风险偏好**：
   - 保守（最大回撤 < 10%）
   - 稳健（最大回撤 < 20%）
   - 进取（最大回撤 < 30%）
3. **规模范围**（默认 5-500 亿）
4. **经理任职年限**（默认 > 2 年）
5. **业绩要求**（默认近 1 年跑赢业绩比较基准）
6. **评级要求**（默认晨星 3 星以上，或无要求）

### 第二步：获取初始基金池

调用 `get_fund_ranking` 获取排行榜作为候选池。
如用户指定类型，传入 `fund_type` 参数过滤。

### 第三步：逐只深度分析

对候选基金（Top 50 或更少）调用以下工具：

1. `get_fund_info` — 获取规模、经理、费率、类型
2. `get_fund_nav_history` — 获取近 1 年净值，计算：
   - 年化收益率
   - 最大回撤 = (峰值 - 谷值) / 峰值
   - 年化波动率 = 日收益标准差 × √252
   - 夏普比率 = (年化收益 - 无风险利率) / 年化波动率
   - 卡玛比率 = 年化收益 / 最大回撤

### 第四步：打分排序

按以下维度打分（满分 100）：

| 维度 | 权重 | 计算方式 |
|------|------|---------|
| 收益风险比 | 30% | 夏普比率排名 |
| 最大回撤 | 25% | 回撤越小分越高 |
| 规模适中 | 15% | 20-100 亿满分，超出扣分 |
| 经理稳定性 | 15% | 任职年限 + 管理产品一致性 |
| 费率水平 | 10% | 总费率（管理费+托管费）越低分越高 |
| 评级加分 | 5% | 晨星 4-5 星加分 |

#### 穿透验证容错

- 持仓明细为空时，仅比较行业配置（申万一级）
- 行业重合度 > 15% 时标注 [⚠️ 行业集中风险]
- 穿透深度为 C 级时（持仓明细为空），综合得分扣 10 分

#### 费率打分容错

- 费率数据完整：正常打分（权重 10%）
- 费率数据缺失：标注 [⚠️ 费率缺失]，该项得分为 0，不参与横向对比
- 同类基金优先推荐费率最低者（数据完整时）

## 注意事项

- 不要一次处理超过 50 只基金的深度分析（太慢）
- 如果候选池超过 50，先用 `get_fund_info` 做一轮粗筛（规模、经理任职），再进入深度分析
- 货币基金和短债基金不适合此筛选框架，应引导用户使用"宝宝类"或"现金管理"工具
- 始终附带免责声明
