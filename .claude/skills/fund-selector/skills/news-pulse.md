# /news-pulse — 快讯快速归因（团队型）

多 Agent 并行归因，10 分钟内输出涨跌原因。

## 触发
"快讯解读"、"为什么涨跌"、"新闻归因"

## 团队结构
| Agent | 覆盖范围 |
|-------|---------|
| 公司事件 Agent | 业绩预告/重组/大股东增减持 |
| 监管政策 Agent | 行业新规/窗口指导/处罚 |
| 行业竞品 Agent | 竞品动态/份额变化 |
| 市场情绪 Agent | 北向资金/融资融券/龙虎榜 |

## 流程
1. 获取涨跌幅异动（±5% 以上）
2. 4 Agent 并行 MCP 搜索
3. Team Lead 汇总归因
4. 生成解读报告

## 输出
- 异动归因（公司/政策/行业/情绪）
- 影响评估 + 后续关注点
