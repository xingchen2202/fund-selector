# /fund-article — 研报写作工厂（团队型）

3 Agent 写作工厂：作者 → 编辑 → 审阅，产出公众号可发文章。

## 触发
"写篇研报"、"公众号文章"、"投资笔记"

## 团队结构
| Agent | 任务 |
|-------|------|
| 作者 Agent | 基于研究数据撰写初稿 |
| 编辑 Agent | 润色排版 + 公众号风格 |
| 审阅 Agent | 读者视角挑刺 + 事实核查 |

## 流程
1. 作者 Agent 读取研究数据（step3/step4 输出）
2. 撰写初稿（含数据引用）
3. 编辑 Agent 润色（标题/段落/排版）
4. 审阅 Agent 抽样验证 15% 数据点
5. 终稿输出

## 输出
- 公众号格式研报（Markdown）
- 数据附录 + 来源标注
