# /fund-team — 多 Agent 投研团队（团队型）

Team Lead 并行调度 4 大师视角 Agent，独立分析后综合研判。

## 触发
"多角度分析 XX"、"投研团队看 XX"、"深度覆盖"

## 团队结构
| Agent | 视角 | 核心问题 |
|-------|------|---------|
| 价值 Agent | 巴菲特 | 财务质量、估值安全边际 |
| 成长 Agent | 段永平 | 商业模式、护城河 |
| 风控 Agent | 李录 | 风险信号、管理质量 |
| 周期 Agent | 芒格 | 行业格局、竞争态势 |

## 流程
1. Team Lead 并行启动 4 Agent
2. 各 Agent 独立 MCP 搜索 + 独立判断
3. 4 视角结论汇总 → 冲突检测（排名差 ≥3）
4. 综合研判 → 生成报告
5. 报告审计门（15% 抽样）

## 输出
- 4 视角独立结论 + 冲突标注
- 综合研判 + 操作建议

## 工具依赖
- `mcp__cn-mutual-fund` — 基金信息/净值/经理/持仓获取
- `mcp__cn-financial` — A股行情/宏观/行业数据
- `tools/financial_rigor.py` — Decimal 精度验算（verify-scale/cross-validate）

## 失败处理
- MCP 超时/异常 → 标注"数据不可用"并继续，不中止整体流程
- MCP 返回陈旧数据（如 2014 年北向资金）→ 标注异常跳过该维度
- 全部 MCP 失败 → 输出"当前无法获取实时数据，请稍后重试"
