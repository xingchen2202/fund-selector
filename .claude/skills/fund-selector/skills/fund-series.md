# /fund-series — 基金系列研报（团队型）

为目标基金/行业生成 8 篇系列研报（约 120k 字，公众号可发）。

## 触发
"XX 行业系列"、"系列研报"、"深度覆盖"

## 系列规划
1. 行业总览：产业链图谱 + 市场规模
2. 上游环节：原材料/设备/技术
3. 中游环节：制造/组装/集成
4. 下游环节：应用/渠道/终端
5. 龙头公司深度
6. 竞争格局：市场份额 + 壁垒
7. 投资逻辑：核心假设 + 催化剂
8. 风险提示：政策/技术/市场

## 流程
1. Team Lead 启动 4 Agent 并行覆盖不同环节
2. 各环节独立研究 + 数据验证
3. 编辑 Agent 润色统一风格
4. 审阅 Agent 质量把关

## 输出
- 8 篇系列文章（每篇 ~15k 字）
- 数据附录 + 来源标注
