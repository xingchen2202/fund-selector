# A股 DCF 模型常见问题排查

## 1. 模型不平衡 / 资产负债表不平

**症状**：资产总计 ≠ 负债总计 + 所有者权益

**常见原因与解决方案**：

| 原因 | 解决方案 |
|------|---------|
| 现金科目未与现金流量表勾稽 | 检查：期末现金 = 期初现金 + 经营现金流 + 投资现金流 + 筹资现金流 |
| 固定资产未扣减折旧 | 检查：期末固定资产 = 期初 + 新增CAPEX - 折旧 - 处置 |
| 未分配利润未更新 | 检查：期末未分配利润 = 期初 + 归母净利润 - 现金分红 |
| 少数股东权益遗漏 | 检查：少数股东权益需随净利润和分红变动 |
| 商誉/无形资产未减值 | 确保减值损失反映在资产端和利润表中 |

## 2. WACC 计算问题

### 中国市场 WACC 参数参考范围

| 参数 | 典型范围 | 说明 |
|------|---------|------|
| 无风险利率 (Rf) | 2.0% – 3.5% | 10年期国债到期收益率 |
| 股权风险溢价 (ERP) | 5.0% – 7.5% | A股市场风险溢价，通常高于美股 |
| Beta | 0.6 – 1.5 | 基于沪深300或中证800回归 |
| 公司特有风险溢价 | 0% – 3% | 小盘股、流动性差的公司需加 |
| 税前债务成本 (Kd) | 3.0% – 6.0% | 参考LPR + 信用利差 |
| 企业所得税率 | 15% / 25% | 高新技术15%，一般25% |
| 目标资本结构 D/(D+E) | 10% – 40% | 因行业而异 |

**常见错误**：
- ❌ 直接使用美国市场的无风险利率和 ERP
- ❌ 用历史 Beta 而未调整（Adjusted Beta = 0.67 × Raw + 0.33 × 1.0）
- ❌ 未考虑 A 股的高新技术企业优惠税率（15%）
- ❌ 使用银行贷款利率而非公司实际加权融资成本

### Ke 计算
```
Ke = Rf + β × ERP + 公司特有风险溢价
```

### WACC 计算
```
WACC = Ke × E/(D+E) + Kd × (1-T) × D/(D+E)
```

## 3. 自由现金流计算错误

**A股常见陷阱**：

| 问题 | 解决方案 |
|------|---------|
| 将经营现金流直接等同于 FCFF | FCFF = EBIT×(1-T) + 折旧摊销 - CAPEX - △NWC |
| 营运资本变动方向搞反 | △NWC 增加 → 现金流出（减少FCFF） |
| CAPEX 遗漏在建工程 | 资本支出应包含固定资产和在建工程的净增加 |
| 忽略非现金项目的资本化研发 | 资本化研发支出应计入 CAPEX |
| 政府补助处理不一致 | 明确政府补助是否属于经常性收入 |

## 4. 终值 (Terminal Value) 问题

### 永续增长法
```
TV = FCF_n × (1 + g) / (WACC - g)
```

**A股参数参考**：
| 参数 | 范围 | 说明 |
|------|------|------|
| 永续增长率 (g) | 2.0% – 3.5% | 不应超过中国GDP长期增速预期 |
| 预测期 | 5 – 10 年 | 成熟行业5年，高成长行业8-10年 |

**常见错误**：
- ❌ 永续增长率设为 5% 以上（不可持续）
- ❌ 终值占比超过 80%（说明近期预测可能过保守或g过高）
- ❌ 预测期末 FCF 仍在高速增长（应趋于稳态）

### 退出倍数法
```
TV = EBITDA_n × 退出倍数
```

退出倍数应参考：
- 行业当前 EV/EBITDA 中位数
- 历史周期的均值水平
- 成熟期合理估值水平

## 5. 敏感性分析检查

必须包含的二维敏感性表：

| 维度1 | 维度2 | 建议范围 |
|--------|--------|---------|
| WACC | 永续增长率 g | WACC ±1.0%，g ±0.5% |
| 营收增速 | 净利率 | 增速 ±5pp，利润率 ±2pp |
| 退出倍数 | WACC | 倍数 ±2x，WACC ±1.0% |

**检查项**：
- [ ] 敏感性表中所有值为正数（否则检查参数极端值）
- [ ] 基准情景在敏感性表的中心位置
- [ ] 不同情景的估值差异在合理范围内（±30%以内）

## 6. 快速验证清单

- [ ] 三表勾稽：资产=负债+权益，现金流与现金余额一致
- [ ] WACC 在合理范围（6%–12%）
- [ ] 永续增长率 < WACC 且 < GDP增速
- [ ] 终值占比 < 75%
- [ ] FCF 增长趋势合理，预测期末趋于稳态
- [ ] 隐含倍数（DCF估值/净利润）与可比公司PE在同一数量级
- [ ] 所有预测年份的关键比率（毛利率、净利率、ROIC）变化有合理解释
- [ ] 已考虑限售股、优先股等对股权价值的稀释
- [ ] 货币单位一致（全部为人民币元或万元）
