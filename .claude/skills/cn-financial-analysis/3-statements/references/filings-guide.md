# 中国上市公司信息披露指南

## 定期报告体系

### 年度报告
- **披露时间**：会计年度结束后 4 个月内（1月1日—4月30日）
- **审计要求**：必须经具有证券期货业务资格的会计师事务所审计
- **关键章节**：
  - 公司简介和主要财务指标
  - 管理层讨论与分析（MD&A）
  - 重要事项
  - 财务报告（三大报表 + 附注）
  - 公司治理
  - 股份变动及股东情况

### 半年度报告
- **披露时间**：上半年结束后 2 个月内（7月1日—8月31日）
- **审计要求**：可以不经审计（但鼓励审计或审阅）
- **内容**：与年报类似但简化

### 季度报告（一季报、三季报）
- **一季报披露**：4月1日—4月30日
- **三季报披露**：10月1日—10月31日
- **审计要求**：无需审计
- **内容**：简要财务数据和经营情况

### 业绩预告与业绩快报
- **业绩预告**：预计净利润出现以下情况时须提前披露：
  - 净利润为负
  - 净利润同比变动 ≥ 50%
  - 扭亏为盈
  - 期末净资产为负
- **业绩快报**：部分交易所要求在年报/半年报正式披露前发布未经审计的初步数据

## 会计准则差异要点

### 中国企业会计准则 (CAS) vs IFRS/US GAAP

| 科目 | CAS | IFRS / US GAAP 对应 |
|------|-----|---------------------|
| 营业收入 | 营业收入 | Revenue |
| 营业成本 | 营业成本 | Cost of Revenue / COGS |
| 销售费用 | 销售费用 | Selling Expenses / S&M |
| 管理费用 | 管理费用 | G&A Expenses |
| 研发费用 | 研发费用 | R&D Expenses |
| 财务费用 | 财务费用（含利息收支） | Interest Expense (net) |
| 其他收益 | 其他收益 | Other Income |
| 投资收益 | 投资收益 | Investment Income |
| 公允价值变动 | 公允价值变动收益 | Fair Value Changes |
| 信用减值 | 信用减值损失 | ECL / Credit Impairment |
| 资产减值 | 资产减值损失 | Asset Impairment |
| 经营活动现金流 | 经营活动产生的现金流量净额 | CFO (Cash from Operations) |

### 关键差异
1. **研发费用资本化**：CAS 与 IFRS 类似，开发阶段可资本化；US GAAP 一般全部费用化
2. **政府补助**：CAS 单独列示"其他收益"或冲减相关费用；IFRS 可抵减资产或确认为收入
3. **非经常性损益**：CAS 有明确定义和披露要求，分析时需重点关注"扣非净利润"
4. **少数股东权益**：CAS 在所有者权益中单独列示
5. **商誉**：CAS 不摊销，每年减值测试；IFRS 类似；US GAAP 2025 后可选择摊销

## 关键监管文件类型

| 代码 | 类型 | 说明 |
|------|------|------|
| 临时公告 | 重大事项 | 股权变动、关联交易、对外担保、重大合同等 |
| 定期报告 | 年报/半年报/季报 | 财务数据和经营状况 |
| 发行文件 | 招股说明书/募集说明书 | IPO或再融资相关文件 |
| 独立意见 | 独立董事意见 | 关联交易等重大事项的独立意见 |
| 法律意见 | 律师事务所出具 | 重大事项合规性意见 |
| 评估报告 | 资产评估报告 | 资产重组、关联交易中的估值 |

## 数据获取路径

### 通过 cn-financial-mcp 获取
- `get_income_statement` → 利润表数据
- `get_balance_sheet` → 资产负债表数据
- `get_cash_flow` → 现金流量表数据
- `get_financial_indicators` → 财务指标（ROE、毛利率等）
- `get_company_info` → 公司基本信息
- `get_company_announcements` → 公司公告

### 补充数据源
- 巨潮资讯网：完整公告原文（PDF）
- 东方财富/同花顺：整理后的财务数据
- Wind/Choice（付费）：更全面的历史数据和一致预期
