# 财务公式参考

## 利润表核心公式

```
毛利润 = 营业收入 - 营业成本
毛利率 = 毛利润 / 营业收入

营业利润 = 毛利润 - 销售费用 - 管理费用 - 研发费用 - 财务费用
         + 其他收益 + 投资收益 + 公允价值变动收益
         - 信用减值损失 - 资产减值损失 + 资产处置收益

利润总额 = 营业利润 + 营业外收入 - 营业外支出

净利润 = 利润总额 - 所得税费用
归母净利润 = 净利润 - 少数股东损益

EPS = 归母净利润 / 加权平均股本数
扣非净利润 = 净利润 - 非经常性损益
扣非 EPS = 扣非净利润 / 加权平均股本数
```

## 资产负债表核心公式

```
资产总计 = 流动资产合计 + 非流动资产合计
负债总计 = 流动负债合计 + 非流动负债合计
所有者权益 = 资产总计 - 负债总计
归母所有者权益 = 所有者权益 - 少数股东权益

流动资产 = 货币资金 + 交易性金融资产 + 应收票据 + 应收账款
         + 预付款项 + 其他应收款 + 存货 + 合同资产 + 其他流动资产

净营运资本 = 流动资产 - 流动负债
有息负债 = 短期借款 + 一年内到期的非流动负债 + 长期借款 + 应付债券

净负债 = 有息负债 - 货币资金 - 交易性金融资产
```

## 现金流核心公式

```
自由现金流 (FCFF) = 经营活动现金流量净额 - 资本支出
                   = 经营活动现金流量净额 - (购建固定资产、无形资产和其他长期资产支付的现金)

股权自由现金流 (FCFE) = FCFF - 利息费用×(1-税率) + 净借款变动

资本支出 (CAPEX) = 购建固定资产等支付的现金 - 处置固定资产等收回的现金
```

## 盈利能力指标

```
毛利率 = 毛利润 / 营业收入
净利率 = 净利润 / 营业收入
归母净利率 = 归母净利润 / 营业收入
扣非净利率 = 扣非净利润 / 营业收入

ROE = 归母净利润 / 平均归母所有者权益
ROA = 净利润 / 平均资产总计
ROIC = NOPAT / 投入资本
     = EBIT × (1-税率) / (归母权益 + 有息负债 - 超额现金)

三费率 = (销售费用 + 管理费用 + 财务费用) / 营业收入
期间费用率 = (销售费用 + 管理费用 + 研发费用 + 财务费用) / 营业收入
```

## 杜邦分析

```
ROE = 净利率 × 总资产周转率 × 权益乘数
    = (净利润/营业收入) × (营业收入/平均总资产) × (平均总资产/平均归母权益)

进一步分解：
ROE = 税收负担率 × 利息负担率 × 营业利润率 × 资产周转率 × 权益乘数
    = (净利润/利润总额) × (利润总额/EBIT) × (EBIT/营业收入)
      × (营业收入/总资产) × (总资产/权益)
```

## 营运能力指标

```
应收账款周转天数 = 365 / (营业收入 / 平均应收账款)
存货周转天数 = 365 / (营业成本 / 平均存货)
应付账款周转天数 = 365 / (营业成本 / 平均应付账款)

现金循环周期 (CCC) = 应收账款周转天数 + 存货周转天数 - 应付账款周转天数
总资产周转率 = 营业收入 / 平均资产总计
```

## 偿债能力指标

```
流动比率 = 流动资产 / 流动负债
速动比率 = (流动资产 - 存货) / 流动负债

资产负债率 = 负债总计 / 资产总计
净负债率 = 净负债 / 归母权益
有息负债率 = 有息负债 / 资产总计

利息保障倍数 = EBIT / 利息费用
经营现金流/有息负债 = 经营活动现金流量净额 / 有息负债
```

## 估值指标

```
PE (市盈率) = 股价 / EPS = 总市值 / 归母净利润
PE-TTM = 总市值 / 最近四个季度归母净利润之和
PB (市净率) = 总市值 / 归母净资产 = 股价 / 每股净资产
PS (市销率) = 总市值 / 营业收入
PEG = PE / 净利润增速(%)

EV = 总市值 + 有息负债 - 货币资金
   = 总市值 + 少数股东权益 + 有息负债 - 现金及等价物
EV/EBITDA = EV / (营业利润 + 折旧 + 摊销)
EV/Sales = EV / 营业收入

股息率 = 每股股利 / 股价
分红率 = 每股股利 / EPS = 现金分红总额 / 归母净利润
```

## 非经常性损益常见项目（A股特色）

```
非经常性损益 = 
  + 非流动资产处置损益
  + 政府补助（计入当期损益的）
  + 委托投资/理财收益
  + 债务重组损益
  + 企业合并成本小于被合并方可辨认净资产公允价值份额的差额
  + 非货币性资产交换损益
  + 因不可抗力因素的资产减值准备转回
  + 其他营业外收支净额
  - 所得税影响额
  - 少数股东权益影响额
```
