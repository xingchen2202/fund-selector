# 三表分析与建模（中国企业会计准则）

## 概述

本技能构建符合中国企业会计准则（CAS）的三大财务报表分析模型。输出为完整 Excel 工作簿，包含历史数据整理、趋势分析、勾稽关系验证和前瞻性预测。

## 可用 MCP 工具

- **`get_income_statement`** — 利润表（按报告期）
- **`get_balance_sheet`** — 资产负债表（按报告期）
- **`get_cash_flow`** — 现金流量表（按报告期）
- **`get_financial_indicators`** — 财务指标（盈利、运营、偿债）
- **`get_company_info`** — 公司基本信息

## 第一部分：利润表（损益表）

### 中国企业会计准则标准科目

```
一、营业收入
  减：营业成本
      税金及附加
      销售费用
      管理费用
      研发费用
      财务费用
        其中：利息费用
              利息收入
      其他收益
  加：投资收益
        其中：对联营/合营企业的投资收益
      公允价值变动收益
      信用减值损失
      资产减值损失
      资产处置收益
二、营业利润
  加：营业外收入
  减：营业外支出
三、利润总额
  减：所得税费用
四、净利润
    其中：归属于母公司股东的净利润
          少数股东损益
五、每股收益
    基本每股收益
    稀释每股收益
```

### 关键比率分析

| 比率 | 公式 | 正常范围 |
|------|------|---------|
| 毛利率 | (营收-营业成本)/营收 | 因行业而异 |
| 销售费用率 | 销售费用/营收 | 消费品 10-30%，工业 3-8% |
| 管理费用率 | 管理费用/营收 | 一般 3-10% |
| 研发费用率 | 研发费用/营收 | 科技 5-20%，传统 1-3% |
| 财务费用率 | 财务费用/营收 | 取决于杠杆 |
| 净利率 | 归母净利/营收 | 因行业而异 |
| 实际税率 | 所得税/利润总额 | 25%（高新 15%） |

```
    应收票据
    应收账款
    应收款项融资
    预付款项
    其他应收款
    存货
    合同资产
    其他流动资产
  非流动资产：
    长期股权投资
    其他权益工具投资
    固定资产
    在建工程
    使用权资产
    无形资产
    商誉
    长期待摊费用
    递延所得税资产
    其他非流动资产
  资产总计

负债：
  流动负债：
    短期借款
    应付票据
    应付账款
    预收款项
    合同负债
    应付职工薪酬
    应交税费
    其他应付款
    一年内到期的非流动负债
    其他流动负债
  非流动负债：
    长期借款
    应付债券
    租赁负债
    长期应付款
    预计负债
    递延收益
    递延所得税负债
    其他非流动负债
  负债合计

所有者权益：
    股本
    资本公积
    其他综合收益
    盈余公积
    未分配利润
    归属于母公司股东权益合计
    少数股东权益
  所有者权益合计
负债和所有者权益总计
```

### 关键比率

| 比率 | 公式 | 说明 |
|------|------|------|
| 资产负债率 | 负债/资产 | 杠杆水平 |
| 流动比率 | 流动资产/流动负债 | 短期偿债 |
| 速动比率 | (流动资产-存货)/流动负债 | 速动偿债 |
| 应收周转天数 | 应收账款×365/营收 | 回款效率 |
| 存货周转天数 | 存货×365/营业成本 | 库存管理 |
| 应付周转天数 | 应付账款×365/营业成本 | 付款节奏 |

## 第三部分：现金流量表

### 中国企业会计准则标准科目

```
一、经营活动产生的现金流量
    销售商品、提供劳务收到的现金
    收到的税费返还
    收到其他与经营活动有关的现金
    购买商品、接受劳务支付的现金
    支付给职工的现金
    支付的各项税费
    支付其他与经营活动有关的现金
  经营活动现金流量净额

二、投资活动产生的现金流量
    收回投资收到的现金
    取得投资收益收到的现金
    处置固定资产等收到的现金净额
    购建固定资产等支付的现金
    投资支付的现金
  投资活动现金流量净额

三、筹资活动产生的现金流量
    吸收投资收到的现金
    取得借款收到的现金
    偿还债务支付的现金
    分配股利、利润支付的现金
  筹资活动现金流量净额

四、现金及现金等价物净增加额
  加：期初现金及现金等价物余额
五、期末现金及现金等价物余额
```

### 现金流质量分析

| 指标 | 公式 | 说明 |
|------|------|------|
| 经营现金流/净利润 | 经营 CF / 归母净利 | >1 说明盈利质量高 |
| 自由现金流 | 经营 CF - CapEx | 可自由支配的现金 |
| CapEx/营收 | 购建固定资产/营收 | 资本密集度 |
| 分红/净利润 | 分红/归母净利 | 分红率 |

## 第四部分：勾稽关系验证

### 三表勾稽检查（必须通过）

1. **资产负债表平衡**：资产 = 负债 + 所有者权益
2. **未分配利润勾稽**：期末未分配利润 = 期初 + 归母净利 - 分红 - 盈余公积提取
3. **现金勾稽**：期末货币资金 ≈ 现金流量表期末余额
4. **折旧勾稽**：固定资产减少 ≈ 利润表折旧 + 处置
5. **利息勾稽**：财务费用 ≈ 有息负债 × 平均利率

## 第五部分：预测建模

### 驱动假设

| 科目 | 驱动因子 |
|------|---------|
| 营收 | 行业增速 × 份额变化，或量 × 价拆分 |
| 毛利 | 毛利率假设 × 营收 |
| 三费 | 费用率假设 × 营收 |
| CapEx | CapEx/营收比率 |
| 折旧 | 固定资产 / 折旧年限 |
| 营运资本 | 周转天数 × 营收或成本 |
| 有息负债 | 资产负债率目标反推 |

### 注意事项
- 预测期 3-5 年
- 所有假设须有来源或逻辑支撑
- 蓝色标注硬编码假设，黑色为公式
- 附注释说明每个关键假设的依据
