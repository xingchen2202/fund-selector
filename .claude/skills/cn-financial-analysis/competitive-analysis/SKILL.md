# 中国市场竞争分析

## 概述

本技能帮助构建全面的中国市场竞争分析框架，覆盖产业链上下游映射、竞争格局评估、政策环境分析和战略定位。适用于投资研究、尽调和战略咨询。

## 可用 MCP 工具

- **`search_stock`** — 搜索同行业上市公司
- **`get_company_info`** — 公司基本信息和主营业务
- **`get_industry_list`** — 申万行业分类列表
- **`get_industry_stocks`** — 行业成分股列表
- **`get_concept_list`** — 概念板块列表
- **`get_financial_indicators`** — 财务指标对比
- **`get_valuation_metrics`** — 估值指标对比
- **`get_income_statement`** — 利润表（分析盈利结构）
- **`get_realtime_quote`** — 实时市值和交易数据

## 工作流

### 第一步：确定分析范围

1. **明确目标公司**和所属行业（申万分类）
2. **确定竞争对手**（通过 MCP 获取行业成分股 + 概念板块）
3. **定义分析维度**：
   - 市场份额
   - 盈利能力
   - 成长性
   - 财务健康度
   - 产业链地位

### 第二步：产业链映射

绘制产业链上下游关系：

```
上游（原材料/零部件）
  │
  ├── 供应商 A（占比 X%）
  ├── 供应商 B（占比 X%）
  │
中游（制造/组装）← 目标公司
  │
  ├── 竞争对手 1（市占率 X%）
  ├── 竞争对手 2（市占率 X%）
  │
下游（渠道/终端客户）
  │
  ├── 经销商/直销
  └── 终端用户群体
```

**分析要点：**
- 上游议价能力（集中度、替代性）
- 下游议价能力（客户集中度、转换成本）
- 行业壁垒（技术、资金、牌照、规模）

### 第三步：竞争格局分析

#### 波特五力模型（中国市场适配版）

| 竞争力 | 评估维度 | A 股特色考量 |
|--------|---------|-------------|
| 现有竞争 | 行业集中度 CR3/CR5/HHI | 国企 vs 民企格局 |
| 新进入者 | 进入壁垒高低 | 行政审批、牌照门槛 |
| 替代品 | 替代威胁程度 | 技术迭代、进口替代 |
| 供应商议价 | 上游集中度 | 资源型行业受政策影响 |
| 买方议价 | 下游集中度 | 政府/国企大客户依赖 |

| 营收增速 | | | | |
| 毛利率 | | | | |

| ROE | | | | |
| 市占率 | | | | |

### 第四步：政策环境分析

**中国特色必须考虑的政策维度：**

1. **产业政策**
   - 国家战略性新兴产业目录
   - "十四五"规划相关政策
   - 产业补贴和税收优惠

2. **监管政策**
   - 行业准入和牌照管理
   - 环保和碳排放约束
   - 反垄断和数据安全

3. **贸易政策**
   - 进出口关税和配额
   - 国产替代和自主可控
   - 中美贸易摩擦影响

4. **地方政策**
   - 区域发展战略
   - 地方政府支持力度
   - 土地和税收优惠

### 第五步：战略定位评估

#### SWOT 分析

| | 有利 | 不利 |
|--|------|------|
| **内部** | 优势(S) | 劣势(W) |
| **外部** | 机会(O) | 威胁(T) |

#### 竞争优势持续性评估

| 竞争优势类型 | 持续性 | 案例 |
|-------------|--------|------|
| 成本优势 | 中 | 规模效应、区位优势 |
| 技术壁垒 | 高 | 专利、know-how |
| 品牌溢价 | 高 | 消费者认知 |
| 网络效应 | 极高 | 平台型企业 |
| 政策壁垒 | 高但有不确定性 | 牌照、资质 |
| 资源独占 | 极高 | 矿权、特许经营 |

## 输出格式

### 报告结构（10-15 页）

1. **执行摘要**（1 页）— 核心结论和竞争定位
2. **产业链图谱**（1-2 页）— 上下游关系和价值分配
3. **竞争格局**（2-3 页）— 波特五力 + 竞争对手矩阵
4. **政策环境**（1-2 页）— 产业/监管/贸易政策
5. **财务对标**（2-3 页）— 关键指标横向比较（附表格）
6. **SWOT 与战略评估**（1-2 页）— 综合定位
7. **结论与建议**（1 页）— 投资/战略含义

- [ ] 产业链映射完整（上中下游）

- [ ] 政策环境覆盖产业/监管/贸易
- [ ] 财务对标数据有 MCP 来源
