# ADR-001: 调用方式选择

## 状态
已采纳

## 背景
fund-weekly-report 需要执行网络请求和文件写入操作，
属于有副作用的操作，需要决定触发方式。

## 决策
使用手动触发（disable-model-invocation: false，user-invocable: true），
用户通过 /fund:report 触发，Claude 也可根据关键词自动识别。

## 原因
1. 有文件写入副作用，需要用户明确意图
2. 网络请求可能耗时，自动触发体验差
3. 手动触发更符合"每周定期检查"的使用场景

## 备选方案
### 方案 A：纯手动（disable-model-invocation: true）
排除原因：用户说"查看持仓"时希望自动触发，不需要每次输入 /fund:report

### 方案 B：完全自动
排除原因：有副作用操作不应自动执行

## 后果
正面：用户口语触发和命令触发都支持
权衡：需要维护触发关键词列表
