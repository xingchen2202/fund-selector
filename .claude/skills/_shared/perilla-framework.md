# 紫苏叶框架（持仓穿透分析标准）

## 概述
紫苏叶视角是对基金持仓底层质量的穿透分析框架，基于基金季报披露的前十大持仓股进行质量评估。

## 分析维度

### 1. 市值风格
- **大盘股**: 市值 > 500 亿
- **中盘股**: 市值 100-500 亿
- **小盘股**: 市值 < 100 亿

### 2. 盈利能力
- **毛利率**: > 30% 为优秀；20-30% 为良好；< 20% 为一般
- **ROE**: > 15% 为优秀；10-15% 为良好；< 10% 为一般

### 3. 行业地位
- **龙头**: 行业市占率前3
- **跟随者**: 行业市占率前10
- **其他**: 市占率前10之外

### 4. 成长性
- **营收增速**: > 20% 为高增长；10-20% 为中增长；< 10% 为低增长

## 综合评分

每只持仓股按以上4个维度打分：
- 优秀 = 3 分
- 良好 = 2 分
- 一般 = 1 分

**总分 ≥ 9**: 符合紫苏叶标准（高质量持仓）
**总分 6-8**: 中等质量
**总分 < 6**: 需关注

## 输出格式

```
[紫苏叶视角]
XX 股份（600XXX）：符合 3 项标准
  - 市值风格：大盘股 ✓
  - 盈利能力：ROE 18.5% ✓
  - 行业地位：行业龙头 ✓
  - 成长性：营收增速 15% △
```

## 注意事项
- 持仓数据来自基金季报，**滞后一个季度**
- 仅分析前十大持仓股
- 不构成投资建议，仅供质量评估参考
