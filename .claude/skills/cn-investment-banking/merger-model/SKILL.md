# 中国并购重组分析

## 概述

本技能构建中国特色的并购重组分析模型，遵循《上市公司重大资产重组管理办法》和相关监管规则。涵盖交易结构设计、估值与定价、业绩承诺分析和监管审批判断。

## 可用 MCP 工具

- **`get_company_info`** — 收购方/标的基本信息
- **`get_income_statement`** — 利润表（增厚/摊薄分析）
- **`get_balance_sheet`** — 资产负债表（合并报表）
- **`get_realtime_quote`** — 当前股价（定价基准）
- **`get_historical_price`** — 历史股价（发行价计算）
- **`get_valuation_metrics`** — 估值参考
- **`get_financial_indicators`** — 财务指标

## 第一部分：交易结构设计

### 常见交易类型

| 类型 | 描述 | 审批要求 |
|------|------|---------|
| 发行股份购买资产 | 换股收购 | 证监会注册 |
| 现金收购 | 直接现金支付 | 交易所审核 |
| 混合支付 | 股份+现金 | 证监会注册 |
| 协议转让 | 大股东转让 | 交易所审核 |
| 要约收购 | 公开要约 | 证监会 |
| 吸收合并 | 一方消灭 | 证监会注册 |

### 重大资产重组认定标准

**触及以下任一即构成重大资产重组：**
- 购买/出售资产总额 ≥ 上市公司资产总额 50%
- 购买/出售资产营收 ≥ 上市公司营收 50%
- 购买/出售资产净资产 ≥ 上市公司净资产 50%

**触及以下构成重组上市（借壳）：**
- 控制权变更 + 注入资产触及上述标准

### 对价支付方式

**发行股份价格：**
- 基准日前 20/60/120 个交易日均价的 80%
- 基准日：董事会决议公告日

**计算方法**（使用 MCP 历史价格数据）：
```
发行价 = max(
    前20日均价 × 80%,
    前60日均价 × 80%,
    前120日均价 × 80%
) 取其一作为底价
```

## 第二部分：估值分析

### 标的估值方法

**中国并购常用方法：**

1. **收益法（DCF）**
   - 中国评估机构普遍使用
   - 参考 cn-dcf-model 技能
   - 通常使用 FCFE 口径

2. **市场法（可比交易/可比公司）**
   - 可比上市公司估值倍数
   - 近期同行业并购案例对价

3. **资产基础法**
   - 资产重置成本
   - 适用于重资产行业

4. **评估增值率分析**

| 项目 | 账面价值 | 评估价值 | 增值率 |
|------|---------|---------|--------|
| 总资产 | | | |
| 净资产 | | | |

**增值率合理性判断**：
- 轻资产公司增值率可较高（技术/品牌价值）
- 重资产公司增值率应较低
- 参考同行业可比交易增值率

## 第三部分：业绩承诺与补偿

### 业绩承诺设计

**监管要求**：
- 发行股份购买资产的交易对手方，须做出业绩承诺
- 承诺期一般 3 年（部分 4 年）
- 承诺指标通常为扣非归母净利润

**业绩承诺表：**

| 年度 | 承诺扣非净利润（万元） | 增速 |
|------|---------------------|------|
| 第1年 | | |
| 第2年 | | |
| 第3年 | | |
| 合计 | | |

### 业绩补偿机制

**补偿方式**：
1. 股份补偿（优先回购注销）
2. 现金补偿
3. 股份+现金混合

**补偿计算：**
```
当年应补偿金额 = (累计承诺净利 - 累计实现净利) / 总承诺净利 × 交易对价 - 已补偿金额
```

**减值测试补偿**：
承诺期末标的资产减值 > 已补偿金额时，需额外补偿。

## 第四部分：增厚/摊薄分析

### 分析框架

| 指标 | 交易前 | 交易后 | 变化 |
|------|--------|--------|------|
| 归母净利润（万元） | | | |
| 总股本（万股） | | | |
| 基本 EPS（元） | | | |
| 加权 ROE | | | |
| 资产负债率 | | | |

**计算关键：**
- 交易后 EPS = (收购方净利 + 标的净利 - 利息损失) / (原股本 + 新发股份)
- 增厚：交易后 EPS > 交易前 EPS
- 摊薄：交易后 EPS < 交易前 EPS

## 第五部分：监管审批

### 审批流程

```
董事会审议通过（停牌/不停牌）
  ↓
独立财务顾问核查
  ↓
股东大会审议（2/3 表决权通过）
  ↓
交易所审核
  ↓
证监会注册（如需）
  ↓
实施过户 + 新股上市
```

### 审核关注点

1. **交易必要性**：为什么要做这笔收购
2. **估值合理性**：评估增值率是否过高
3. **业绩承诺可实现性**：承诺利润是否合理
4. **标的独立性**：关联交易、同业竞争
5. **整合方案**：收购后如何管理
6. **信息披露充分性**：是否存在隐瞒

## 输出格式

### 并购分析报告

1. **交易概览**（1 页）— 各方基本情况 + 交易方案摘要
2. **标的分析**（3-5 页）— 标的业务、财务、估值
3. **交易方案**（2-3 页）— 对价、支付方式、发行价格
4. **业绩承诺**（1-2 页）— 承诺设计 + 补偿机制
5. **增厚/摊薄分析**（1-2 页）— EPS/ROE 影响
6. **监管判断**（1-2 页）— 审批路径 + 关注点
7. **风险提示**（1 页）

**交付物**：
- DOCX 分析报告
- XLS 并购模型（估值 + 增厚/摊薄 + 敏感性）
