# 中国A股并购重组监管规则

## 重大资产重组认定标准

### 触发条件（任一满足即构成重大资产重组）

| 指标 | 标准 |
|------|------|
| **资产总额** | 购买/出售资产总额占上市公司最近一个会计年度经审计的合并财务会计报告期末资产总额的比例 ≥ 50% |
| **营业收入** | 购买/出售资产在最近一个会计年度所产生的营业收入占上市公司同期经审计的合并财务会计报告营业收入的比例 ≥ 50% |
| **资产净额** | 购买/出售资产净额占上市公司最近一个会计年度经审计的合并财务会计报告期末净资产额的比例 ≥ 50%，且超过 5,000 万元 |

### 计算口径
- **资产总额**：标的资产账面值和成交金额的较高者
- **营业收入**：标的资产在最近一个会计年度所产生的营业收入
- **资产净额**：标的资产净资产额与成交金额的较高者

## 审核程序

### 无需审核（信息披露）
- 不构成重大资产重组
- 构成重大资产重组但符合简易/快速审核情形（以当期规则为准）

### 简易/快速审核（以当期规则为准）
- 是否适用简易/快速审核，由交易规模、交易结构、是否涉及发行股份、是否构成重组上市等因素综合判断。
- 具体适用条件以最新《上市公司重大资产重组管理办法》及交易所规则为准。

### 交易所审核（重组委）
- 构成重大资产重组且不满足小额快速条件
- 构成重组上市（借壳）

### 证监会注册程序
- 重组上市（借壳）
- 涉及发行股份购买资产且需履行注册程序的重组交易

## 重组上市（借壳）标准

### 认定标准
上市公司自控制权发生变更之日起 36 个月内，向收购人及其关联人购买资产，导致以下任一情形：
1. 购买的资产总额 ≥ 上市公司控制权发生变更前一个会计年度经审计的合并财务会计报告期末资产总额的 100%
2. 购买的资产在最近一个会计年度所产生的营业收入 ≥ 上市公司控制权发生变更前一个会计年度经审计的合并财务会计报告营业收入的 100%
3. 购买的资产净额 ≥ 上市公司控制权发生变更前一个会计年度经审计的合并财务会计报告期末净资产额的 100%
4. 为购买资产发行的股份占上市公司首次向收购人及其关联人购买资产的董事会决议前一个交易日的股份的比例 ≥ 100%

### 借壳上市条件
需同时满足：
1. 符合 IPO 条件（财务指标、持续经营、公司治理等）
2. 购买资产对应经营实体应符合现行首发与上市的相关规则要求
3. 上市公司及其控股股东、实际控制人最近 3 年不存在重大违法行为

## 发行股份购买资产定价

### 定价基准日
- 董事会决议公告日
- 股东大会决议公告日
- 发行期首日（可转换公司债券转股价格调整）

### 发行价格
- **不低于定价基准日前 20 个交易日**公司股票交易均价的 80%
- 定价基准日前 20 个交易日股票交易均价 = 定价基准日前 20 个交易日股票交易总额 / 定价基准日前 20 个交易日股票交易总量

### 锁定期
| 交易对方 | 锁定期 |
|---------|--------|
| 上市公司控股股东、实际控制人及其控制的关联人 | 36个月 |
| 通过认购本次发行的股份取得上市公司实际控制权的投资者 | 36个月 |
| 取得本次发行的股份时，对其用于认购股份的资产持续拥有权益的时间不足 12 个月 | 36个月 |
| 其他交易对方 | 12个月 |

## 业绩承诺与补偿

### 强制业绩承诺
以下情况必须进行业绩承诺：
1. 上市公司向控股股东、实际控制人或者其控制的关联人购买资产
2. 导致控制权发生变更的资产购买
3. 向关联人购买资产

### 业绩承诺期
- 通常为 3 年（重组完成后连续 3 个会计年度）
- 可延长至 4-5 年

### 补偿方式
- **股份补偿**：按比例回购注销
- **现金补偿**：现金支付
- **股份+现金**：组合补偿

### 补偿计算公式
```
补偿金额 = (承诺净利润 - 实际净利润) × 补偿系数
补偿股份数 = 补偿金额 / 发行价格
```

## 关联交易监管

### 关联方认定
- 控股股东、实际控制人及其控制的关联方
- 持有上市公司 5% 以上股份的股东及其关联方
- 上市公司董事、监事、高级管理人员及其关联方

### 关联交易审核
- 关联交易需独立董事发表意见
- 需股东大会审议（关联股东回避表决）
- 需评估机构出具评估报告
- 需财务顾问发表意见

## 国资监管（如涉及）

### 审批层级
| 维度 | 要点 |
|------|------|
| 产权归属层级 | 按中央企业/地方国企及授权范围确定审批主体 |
| 交易事项类型 | 涉及产权转让、增资、资产评估备案/核准等分别适用不同程序 |
| 金额与权限 | 金额阈值及权限划分并非全国统一，按主管国资监管机构制度执行 |

### 评估要求
- 必须由具有相应资质的评估机构评估
- 评估结果需报国资委备案或核准
- 交易价格不得低于评估结果的 90%

## 常见问题

### Q: 重大资产重组与借壳上市的区别？
**A:** 
- **重大资产重组**：资产规模达到一定比例，需交易所/证监会审核
- **借壳上市**：控制权变更 + 重大资产重组，需符合 IPO 条件

### Q: 发行股份购买资产 vs 现金收购？
**A:** 
- **发行股份**：不消耗现金，但会稀释股权，需审核
- **现金收购**：不稀释股权，但需资金，通常无需审核（除非构成重大资产重组）

### Q: 业绩承诺未完成怎么办？
**A:** 
- 触发补偿机制（股份回购或现金补偿）
- 可能影响后续再融资
- 严重情况下可能被监管处罚
