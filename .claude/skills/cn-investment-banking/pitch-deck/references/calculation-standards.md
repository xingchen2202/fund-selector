# 中国投行路演材料计算标准

## 估值计算

### 市值计算
```
总市值 = 股价 × 总股本
流通市值 = 股价 × 流通股本

注意：
- A股存在限售股，总市值 ≠ 流通市值
- IPO 定价时使用发行后总股本
- 并购估值时需考虑是否含非上市资产
```

### 企业价值 (EV)
```
EV = 总市值 + 少数股东权益 + 有息负债 - 现金及等价物
   = 总市值 + 净负债 + 少数股东权益

有息负债 = 短期借款 + 一年内到期的非流动负债 + 长期借款 + 应付债券
现金及等价物 = 货币资金 + 交易性金融资产（短期理财）
```

### 常用倍数
```
PE = 总市值 / 归母净利润
PE-TTM = 总市值 / 最近四个季度归母净利润
PB = 总市值 / 归母净资产
PS = 总市值 / 营业收入
EV/EBITDA = EV / (营业利润 + 折旧 + 摊销)
EV/Sales = EV / 营业收入
```

### EBITDA 计算（中国准则）
```
EBITDA = 营业利润 + 折旧费用 + 摊销费用
       = 营业收入 - 营业成本 - 三费 + 折旧 + 摊销

注意：
- 财务费用中的利息收入应加回
- 投资收益通常不计入
- 政府补助可选择计入或剔除，需一致
```

## 财务指标计算

### 增长率
```
YoY 增长率 = (本期 - 去年同期) / |去年同期| × 100%
CAGR = (期末值/期初值)^(1/n) - 1

注意：
- 去年同期为负数时，增长率需用绝对值计算
- 去年同期为零时，标注"N/M"（Not Meaningful）
- 扭亏为盈时标注"扭亏"
```

### 利润率
```
毛利率 = (营业收入 - 营业成本) / 营业收入 × 100%
EBITDA 利润率 = EBITDA / 营业收入 × 100%
EBIT 利润率 = 营业利润 / 营业收入 × 100%
净利率 = 净利润 / 营业收入 × 100%
归母净利率 = 归母净利润 / 营业收入 × 100%
```

### 回报率
```
ROE = 归母净利润 / 平均归母净资产 × 100%
ROA = 净利润 / 平均总资产 × 100%
ROIC = NOPAT / 投入资本
NOPAT = EBIT × (1 - 有效税率)
投入资本 = 归母权益 + 有息负债 - 超额现金
```

## 并购特有计算

### 交易倍数
```
交易 PE = 交易对价 / 标的净利润
交易 PB = 交易对价 / 标的净资产
交易 PS = 交易对价 / 标的营收
交易 EV/EBITDA = (交易对价 + 承担负债 - 现金) / 标的 EBITDA
```

### 溢价率
```
控制权溢价 = (交易价格 / 交易前股价 - 1) × 100%
估值溢价 = (交易 PE / 行业 PE 中位数 - 1) × 100%
```

### 摊薄/增厚分析
```
增厚率 = (合并后 EPS - 收购方原 EPS) / 收购方原 EPS × 100%
> 0: 增厚（accretive）
< 0: 摊薄（dilutive）
```

## 数据精度规范

| 数据类型 | 精度要求 | 示例 |
|---------|---------|------|
| 市值/交易金额 | 亿元，2位小数 | 1,234.56亿元 |
| 每股指标 | 元，2位小数 | 12.34元 |
| 倍数 | 1位小数 + "x" | 15.2x |
| 百分比 | 1位小数 + "%" | 12.3% |
| 增速 | 1位小数 + "%" | 25.6% |
| 金额（表格） | 百万元，整数 | 12,345 |

| FY2024A / 2024A | 2024财年实际 |

| NTM | 未来12个月（Next Twelve Months） |
| TTM | 过去12个月（Trailing Twelve Months） |
| 1H24 / H1 2024 | 2024年上半年 |
| 3Q24 / Q3 2024 | 2024年第三季度 |
