# 路演 PPT 页面模板

## 模板一：执行摘要页

### 适用场景
IPO路演、并购推介、投资价值分析的首页或概要页。

### 布局
```
┌──────────────────────────────────────────────────┐
│ [公司名称] — 投资亮点                              │
│ ─────────────────────────────────────────────── │
│                                                  │
│  ┌─────────┐  ┌─────────┐  ┌─────────┐         │
│  │ 亮点 1   │  │ 亮点 2   │  │ 亮点 3   │         │
│  │ 行业龙头  │  │ 高成长   │  │ 技术壁垒  │         │
│  │ [2-3行   │  │ [2-3行   │  │ [2-3行   │         │
│  │  描述]   │  │  描述]   │  │  描述]   │         │
│  └─────────┘  └─────────┘  └─────────┘         │
│                                                  │
│  ┌───────────────────────────────────────┐      │
│  │ 关键财务数据：                          │      │
│  │ 2024营收 XX亿 | 净利 XX亿 | ROE XX%   │      │
│  │ 2025E PE XX.Xx | 目标市值 XXX亿        │      │
│  └───────────────────────────────────────┘      │
│                                                  │
│ 来源：公司公告                              [P.3] │
└──────────────────────────────────────────────────┘
```

│  │                                              │ │
│  │  [收购方]  ──(XX亿元/X.XX亿股)──→ [标的方]   │ │
│  │                                              │ │
│  │  交易方式：发行股份 + 支付现金                  │ │
│  │  交易对价：XX.XX 亿元                         │ │
│  │  发行价格：XX.XX 元/股                        │ │

│                                                  │

## 使用说明

1. 根据项目类型选择合适的模板组合
2. 每页只传达一个核心信息
3. 数据图表 > 纯文字（视觉优先）
4. 保持整套材料风格统一
5. 标题应是洞察/结论，而非主题描述
