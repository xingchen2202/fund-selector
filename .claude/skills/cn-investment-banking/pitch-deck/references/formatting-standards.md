# 中国投行路演材料格式标准

## 页面设置

### PPT 规格
- **页面比例**：16:9（宽屏）
- **页面尺寸**：33.87cm × 19.05cm
- **边距**：上下左右各 1.5cm
- **页码**：右下角，灰色小字

### 字体规范

| 用途 | 中文字体 | 英文字体 | 字号 |
|------|---------|---------|------|
| 封面标题 | 微软雅黑 Bold | Arial Bold | 28-32pt |
| 章节标题 | 微软雅黑 Bold | Arial Bold | 22-26pt |
| 页面标题 | 微软雅黑 Bold | Arial Bold | 18-20pt |
| 正文 | 微软雅黑 Regular | Arial | 12-14pt |
| 表格内容 | 微软雅黑 Regular | Arial | 10-12pt |
| 脚注/来源 | 微软雅黑 Light | Arial | 8-9pt |
| 数字/财务数据 | — | Arial / Calibri | 10-12pt |

### 颜色方案

**主色调（选择一组）：**

方案A - 深蓝专业风：
```
主色:   #003366 (深蓝)   - 标题、强调
辅色1: #0066CC (中蓝)   - 副标题、图表主色
辅色2: #66AADD (浅蓝)   - 背景色块、辅助

方案B - 红金大气风：
```

## 标题撰写规范

### ❌ 不好的标题（描述性）
- "公司财务数据"
- "行业概况"
- "估值分析"

### ✅ 好的标题（洞察性）
- "公司近三年营收 CAGR 达 25%，远超行业均值"
- "行业集中度快速提升，CR5 从 30% 升至 52%"
- "当前估值处于历史 25 分位，具备显著安全边际"

## 表格格式

### 财务数据表
```
┌────────────┬──────┬──────┬──────┬──────┬──────┐
│ 指标        │2022A │2023A │2024A │2025E │2026E │
├────────────┼──────┼──────┼──────┼──────┼──────┤
│ 营业收入    │  XXX │  XXX │  XXX │  XXX │  XXX │  ← 历史黑色
│  YoY       │ XX%  │ XX%  │ XX%  │ XX%  │ XX%  │  ← 预测蓝色
│ 毛利率      │XX.X% │XX.X% │XX.X% │XX.X% │XX.X% │
│ 净利润      │  XXX │  XXX │  XXX │  XXX │  XXX │
│  YoY       │ XX%  │ XX%  │ XX%  │ XX%  │ XX%  │
├────────────┼──────┼──────┼──────┼──────┼──────┤
│ EPS (元)   │ X.XX │ X.XX │ X.XX │ X.XX │ X.XX │
│ PE (x)     │ XX.X │ XX.X │ XX.X │ XX.X │ XX.X │
└────────────┴──────┴──────┴──────┴──────┴──────┘
```

### 格式要点
- 历史数据：黑色，居右对齐
- 预测数据：蓝色，居右对齐
- 合计行：加粗 + 上下边框
- 负数：红色 或 括号
- 千分位分隔符
- 表头：深色底 + 白字

## 图表规范

### 通用规则
- 每张图表一个核心信息
- 标题在图表上方
- 图例在图表内部或上方
- 数据来源在图表下方
- 使用统一颜色方案

### 数值标签
- 柱状图：标注在柱顶
- 折线图：关键转折点标注
- 饼图：标注百分比和绝对值

## 页面布局建议

| 内容类型 | 推荐布局 |
|---------|---------|
| 关键要点 | 左文右图 |
| 财务数据对比 | 全幅表格 |
| 时间序列趋势 | 全幅图表 |
| 公司 vs 竞对 | 左右对比 |
| 流程/逻辑 | 自上而下流程图 |
| 多维对比 | 2×2 矩阵 |
