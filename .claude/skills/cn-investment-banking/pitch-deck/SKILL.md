# 中国投行路演材料

## 概述

本技能帮助创建专业的中国投行路演 PPT 材料，适用于 IPO 路演、并购推介、定增推介和债券发行等场景。输出为 PowerPoint 格式，遵循国内投行通用排版标准。

## 可用 MCP 工具

- **`get_company_info`** — 公司基本信息
- **`get_income_statement`** — 财务数据
- **`get_balance_sheet`** — 资产数据
- **`get_financial_indicators`** — 关键指标
- **`get_valuation_metrics`** — 估值数据
- **`get_industry_stocks`** — 行业对标
- **`get_realtime_quote`** — 最新行情

## IPO 路演材料结构

### 封面（1 页）
- 公司 LOGO 和全称
- "首次公开发行股票并在[板块]上市"
- "投资价值分析报告"
- 保荐机构名称和 LOGO
- 日期

### 目录（1 页）

### 一、投资亮点（2-3 页）

**核心投资逻辑（3-5 条）：**
每条用一句话 + 1-2 个数据支撑

示例：
- 🏆 行业龙头地位 — 市占率 XX%，排名第 X
- 📈 高成长性 — 近 3 年营收 CAGR XX%
- 💰 优秀盈利能力 — 毛利率 XX%，ROE XX%
- 🔬 强大研发实力 — 研发投入占比 XX%，专利 XXX 项
- 🌐 广阔市场空间 — 行业规模 XXX 亿，预计增速 XX%

### 二、公司介绍（3-5 页）

1. **公司概况**
   - 成立时间、注册地、员工数
   - 股权结构图
   - 主营业务一览

2. **发展历程**
   - 关键里程碑时间轴

3. **管理团队**
   - 核心高管简历
   - 技术团队背景

### 三、行业分析（4-6 页）

1. **行业概况**
   - 市场规模和增速
   - 增长驱动因素

2. **竞争格局**
   - 市场份额分布（饼图）
   - 公司定位

3. **产业政策**
   - 关键支持政策
   - 政策红利

4. **行业趋势**
   - 技术发展方向
   - 消费/需求变化

### 四、业务分析（4-6 页）

1. **商业模式**
   - 价值创造链条
   - 盈利模式

2. **核心竞争力**
   - 技术优势
   - 规模优势
   - 品牌/渠道优势

3. **产品/服务**
   - 产品矩阵
   - 核心产品市场地位

4. **客户与渠道**
   - 客户结构
   - 销售网络

### 五、财务分析（3-5 页）

1. **收入分析**
   - 营收趋势（柱状图）
   - 收入结构（饼图）
   - 增长驱动

2. **盈利分析**
   - 利润趋势
   - 利润率趋势（折线图）
   - 与同行对比

3. **关键指标摘要表**

| 指标 | 2022A | 2023A | 2024A |
|------|-------|-------|-------|
| 营收（亿元） | | | |
| 归母净利（亿元） | | | |
| 毛利率 | | | |
| 净利率 | | | |
| ROE | | | |

### 六、募集资金运用（1-2 页）

| 项目 | 金额（亿元） | 用途说明 |
|------|-------------|---------|
| 项目 A | | |
| 项目 B | | |
| 补充流动资金 | | |
| 合计 | | |

### 七、估值分析（2-3 页）

1. **可比公司估值**
2. **估值结论**
3. **发行价格区间**

### 尾页（1 页）
- "感谢聆听"
- 联系方式

## 格式规范

### PPT 设计标准

- **尺寸**：16:9 宽屏
- **字体**：微软雅黑/思源黑体
- **主色调**：深蓝系（#003366 或 #1A3C6E），辅以金色/红色
- **页码**：右下角
- **LOGO**：每页右上角
- **图表风格**：简洁、配色一致、有数据标签

### 数据展示原则

- 每页一个核心信息
- 图表优于表格，表格优于纯文字
- 关键数据加粗/高亮
- 来源标注在页脚

**示例**：`某科技公司_IPO路演_20250315.pptx`
