# 中国A股IPO上市条件

## 主板（上交所/深交所主板）

### 财务指标要求（三选一）

| 标准 | 要求 |
|------|------|
| **标准一** | 最近3年净利润均为正，累计净利润 ≥ 1.5亿元；最近一年净利润 ≥ 6,000万元；最近3年经营活动现金流量净额累计 ≥ 1亿元 或 营业收入累计 ≥ 10亿元 |
| **标准二** | 预计市值 ≥ 50亿元，最近一年净利润为正，营业收入 ≥ 6亿元 |
| **标准三** | 预计市值 ≥ 80亿元，最近一年净利润为正，营业收入 ≥ 8亿元 |

### 其他要求
- 持续经营时间 ≥ 3年
- 最近3年主营业务、董事、高管无重大变化
- 最近3年实际控制人未发生变更
- 股权清晰，无重大权属纠纷
- 最近3年无重大违法行为

## 科创板（上交所）

### 财务指标要求（五选一）

| 标准 | 要求 |
|------|------|
| **标准一** | 预计市值 ≥ 10亿元，最近2年净利润均为正且累计 ≥ 5,000万元；或最近1年净利润为正且营业收入 ≥ 1亿元 |
| **标准二** | 预计市值 ≥ 15亿元，最近1年营业收入 ≥ 2亿元，最近3年研发投入占营业收入比例累计 ≥ 15% |
| **标准三** | 预计市值 ≥ 20亿元，最近1年营业收入 ≥ 3亿元，最近3年经营活动现金流量净额累计 ≥ 1亿元 |
| **标准四** | 预计市值 ≥ 30亿元，最近1年营业收入 ≥ 3亿元 |
| **标准五** | 预计市值 ≥ 40亿元，主要业务或产品需经国家有关部门批准，市场空间大，已取得阶段性成果 |

### 行业定位
- 新一代信息技术
- 高端装备
- 新材料
- 新能源
- 节能环保
- 生物医药
- 符合科创板定位的其他领域

### 其他要求
- 持续经营时间 ≥ 3年
- 最近2年实际控制人未发生变更
- 最近3年无重大违法行为
- 符合科创板定位

## 创业板（深交所）

### 财务指标要求（三选一）

| 标准 | 要求 |
|------|------|
| **标准一** | 最近2年净利润均为正，累计净利润 ≥ 5,000万元 |
| **标准二** | 预计市值 ≥ 10亿元，最近1年净利润为正且营业收入 ≥ 1亿元 |
| **标准三** | 预计市值 ≥ 50亿元，最近1年营业收入 ≥ 3亿元 |

### 行业定位
- 符合"三创四新"（创新、创造、创意，新技术、新产业、新业态、新模式）
- 传统行业与新技术、新产业、新业态、新模式深度融合

### 其他要求
- 持续经营时间 ≥ 3年
- 最近2年实际控制人未发生变更
- 最近3年无重大违法行为

## 北交所（北京证券交易所）

### 财务指标要求（四选一）

| 标准 | 要求 |
|------|------|
| **标准一** | 预计市值 ≥ 2亿元，最近2年净利润均 ≥ 1,500万元且加权平均ROE ≥ 8%，或最近1年净利润 ≥ 2,500万元且加权平均ROE ≥ 8% |
| **标准二** | 预计市值 ≥ 4亿元，最近2年平均营业收入 ≥ 1亿元，最近1年营业收入增长率 ≥ 30%，最近1年经营活动现金流量净额为正 |
| **标准三** | 预计市值 ≥ 8亿元，最近1年营业收入 ≥ 2亿元，最近2年研发投入合计占营业收入比例 ≥ 8% |
| **标准四** | 预计市值 ≥ 15亿元，最近2年研发投入合计 ≥ 5,000万元 |

### 其他要求
- 在新三板连续挂牌满12个月（创新层）
- 符合北交所定位（创新型中小企业）
- 最近3年无重大违法行为

## 注册制流程

### 主要阶段

| 阶段 | 时间 | 关键事项 |
|------|------|---------|
| **辅导备案** | 3-6个月 | 券商辅导，地方证监局备案 |
| **申报受理** | T+0 | 向交易所提交申请文件 |
| **问询回复** | T+20日起 | 交易所多轮问询，发行人回复 |
| **上市委审议** | 问询通过后 | 上市委会议审议 |
| **注册** | 上市委通过后 | 证监会注册 |
| **发行** | 注册后 | 询价、定价、申购、上市 |

### 关键时间节点
- **受理后20个工作日内**：交易所首轮问询
- **问询回复后**：交易所决定是否继续问询或提交上市委
- **上市委审议通过后**：交易所报送证监会注册
- **证监会注册决定**：20个工作日内（可延长至60个工作日）

## 发行定价

### 询价对象
- 证券公司、基金管理公司、信托公司、财务公司、保险公司、合格境外机构投资者、私募基金管理人等专业机构投资者

### 定价方式
- **网下询价**：专业机构投资者报价，按当期发行承销规则对高报价进行剔除并确定有效报价区间后定价
- **网上定价**：向公众投资者发行，价格与网下询价结果一致
- **战略配售**：向战略投资者配售，锁定期12-36个月

### 定价限制
- 发行价格不得低于每股面值（并应符合当期发行承销规则）
- 主板发行市盈率不设统一硬上限，定价约束以当期监管与发行承销规则为准
- 科创板/创业板/北交所：市场化定价，无市盈率上限

## 常见问题

### Q: 主板、科创板、创业板、北交所如何选择？
**A:** 
- **主板**：成熟企业，盈利稳定
- **科创板**：硬科技、高研发投入
- **创业板**：成长型创新企业
- **北交所**：专精特新中小企业

### Q: 注册制下审核重点是什么？
**A:** 
- 信息披露的真实性、准确性、完整性
- 是否符合板块定位
- 持续经营能力
- 公司治理规范性

### Q: 发行失败的风险？
**A:** 
- 询价阶段：有效报价不足或价格过低
- 申购阶段：有效申购不足（需回拨机制）
- 上市后：破发风险（需关注市场环境）
