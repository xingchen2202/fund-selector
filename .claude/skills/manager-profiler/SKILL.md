# 基金经理画像

系统化分析基金经理的投资风格、能力圈和风格稳定性。选主动基金本质上就是选人。

## 触发条件

- "基金经理" / "经理画像" / "选人"
- "XX 经理怎么样" / "介绍一下 XX"
- "张坤是什么风格" / "葛兰靠谱吗"

## 工作流

### 第一步：获取经理基本信息

调用 `get_fund_manager_info`，传入经理姓名或基金代码。

获取：姓名、公司、从业年限、现任基金列表、管理总规模、最佳回报。

### 第二步：获取各产品详情

对经理管理的每只基金调用 `get_fund_info`，获取：
- 基金类型（股/债/混）
- 业绩比较基准
- 费率结构

### 第三步：风险收益分析

对每只基金调用 `get_fund_nav_history`（近 1 年），计算：
- 年化收益率
- 年化波动率
- 最大回撤
- 夏普比率
- 与业绩比较基准的超额收益

### 第四步：风格分类

根据以下维度判断经理风格：

#### 市值风格
- **大盘**：重仓股平均市值 > 5000 亿
- **中盘**：重仓股平均市值 500-5000 亿
- **小盘**：重仓股平均市值 < 500 亿
- **灵活**：无明显倾向

#### 估值风格（需结合持仓分析）
- **深度价值**：平均 PE < 15，PB < 1.5
- **价值**：平均 PE < 25，PB < 2.5
- **均衡**：PE 15-35
- **成长**：平均 PE > 30，看重营收增速
- **高成长**：PE > 50，关注赛道空间

#### 行业集中度
- **集中派**：前三大行业占股票仓位 > 60%
- **均衡型**：前三大行业占 40-60%
- **分散型**：前三大行业 < 40%

#### 特殊标签
- **画线派**：净值曲线平滑，回撤控制极佳
- **赛道型**：重仓单一高景气行业（如新能源、AI）
- **轮动型**：行业配置随市场变化较快
- **逆向型**：偏好低估值、冷门标的

### 第五步：风格漂移检测

对比经理在不同时期（近 1 年 vs 近 3 年）的持仓变化：
- 市值风格是否变化（大盘 → 小盘？）
- 行业配置是否漂移（消费 → 科技？）
- 换手率是否显著变化

## 注意事项

- 如果经理管理产品超过 10 只，只分析规模最大的 5 只 + 业绩最好的 2 只
- 新任职经理（< 1 年）应提示"历史数据较短，判断需谨慎"
- 被动指数基金的经理不需要做风格分析，直接告知用户
- 始终附带免责声明
