# 投资逻辑跟踪

系统化跟踪 A 股投资标的的 **核心投资逻辑**，验证关键假设，追踪催化剂事件，及时发现预期差。

## 可用 MCP 工具

- **`get_realtime_quote`** — 实时行情监控
- **`get_income_statement`** — 财务数据验证
- **`get_financial_indicators`** — 关键指标变化
- **`get_stock_news`** — 最新新闻和公告
- **`get_company_announcements`** — 公司公告
- **`get_valuation_metrics`** — 估值变化

---

## 工作流

### 第一步：建立投资逻辑档案

对每个跟踪标的建立结构化档案：

```
【投资逻辑卡片】
━━━━━━━━━━━━━━━━━━━━
标的：[名称]（[代码]）
建仓日期：[日期]
建仓价格：[价格]
当前价格：[价格]
涨跌幅：[X%]

评级：买入 / 增持 / 中性
目标价：[价格]（上涨空间 X%）

核心投资逻辑（3-5 条）：
1. [逻辑 1] — 状态：✅验证 / ⏳待验证 / ❌证伪
2. [逻辑 2] — 状态：...
3. [逻辑 3] — 状态：...

关键假设：
- 营收增速假设：[X%] → 实际：[Y%]
- 毛利率假设：[X%] → 实际：[Y%]
- 市占率假设：[X%] → 实际：[Y%]
━━━━━━━━━━━━━━━━━━━━
```

### 第二步：催化剂日历

建立并维护催化剂事件日历：

| 日期 | 事件 | 类型 | 影响预判 | 实际结果 |
|------|------|------|---------|---------|
| | 一季报披露 | 财报 | | |
| | 限售解禁 | 供给 | | |
| | 行业政策发布 | 政策 | | |
| | 股东大会 | 治理 | | |
| | 新产品发布 | 业务 | | |
| | 定增/回购 | 资本 | | |

**A 股关键催化剂类型：**
1. **财报催化** — 季报/年报/业绩预告/快报
2. **政策催化** — 产业政策/监管变化
3. **资本催化** — 定增/回购/增减持/解禁
4. **业务催化** — 新产品/新客户/新订单/新产能
5. **行业催化** — 行业景气拐点/价格变动
6. **市场催化** — 纳入指数/北向资金/分析师覆盖

### 第三步：预期差跟踪

**核心理念**：股价由预期差驱动，而非绝对好坏。

| 指标 | 市场预期 | 我方预期 | 实际值 | 预期差 |
|------|---------|---------|--------|--------|
| Q1 营收 | | | | |
| Q1 归母净利 | | | | |
| 毛利率 | | | | |
| 新签订单 | | | | |
| 产品价格 | | | | |

**预期差分级：**
- **正向超预期** — 实际 > 市场预期 > 我方预期 → 强烈看多
- **符合预期** — 实际 ≈ 市场预期 ≈ 我方预期 → 维持判断
- **低于预期** — 实际 < 市场预期 → 评估是否证伪

### 第四步：定期复盘

**每周/每月复盘模板：**

```
【投资逻辑复盘 — [日期]】

一、股价表现
   当期涨跌幅：X%
   同期基准（沪深300）：Y%
   超额收益：X-Y%

二、逻辑验证更新
   逻辑1：[更新说明]
   逻辑2：[更新说明]
   逻辑3：[更新说明]

三、新信息/新变量
   - [重要公告/新闻]
   - [行业变化]
   - [竞争对手动态]

四、预期修正
   营收预测：[原值] → [修正值]
   净利预测：[原值] → [修正值]
   目标价：[原值] → [修正值]

五、操作建议
   维持 / 加仓 / 减仓 / 清仓
   理由：[简述]
```
