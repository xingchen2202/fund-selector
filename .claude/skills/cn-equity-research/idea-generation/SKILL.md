# A 股选股策略

## 工作流

### 第一步：确定筛选条件

询问用户筛选参数：
- **方向**：做多 / 做空 / 双向
- **市值**：大盘（>500亿）、中盘（100-500亿）、小盘（<100亿）
- **行业**：申万一级行业 / 跨行业
- **风格**：价值、成长、质量、事件驱动、主题投资
- **板块**：主板 / 科创板 / 创业板 / 北交所

### 第二步：量化筛选

**根据风格运行不同筛选器：**

#### 价值筛选
- PE-TTM < 行业中位数
- PB < 历史均值 -1 标准差
- 股息率 > 3%
- 自由现金流为正
- ROE > 10%
- 资产负债率 < 60%

#### 成长筛选
- 营收增速 > 20%（连续 2 年）
- 归母净利增速 > 25%
- 毛利率稳定或扩张
- 研发费用率上升
- 行业增速 > GDP 增速
- PEG < 1

#### 质量筛选
- ROE > 15%（连续 3 年）
- 经营现金流/净利润 > 0.8
- 资产负债率稳定
- 扣非净利/归母净利 > 0.8
- 商誉/净资产 < 20%
- 无财务造假历史

#### 事件驱动
- 业绩预告超预期（预增/扭亏）
- 限售股解禁日前后
- 定增/回购/增持公告
- 并购重组进展
- 纳入/调出指数

### 第三步：A 股特色信号

**使用 MCP 工具获取以下数据：**

1. **北向资金**
   - `get_market_overview` — 沪深港通资金流向
   - 重点关注：连续净买入/净卖出

2. **龙虎榜**
   - 关注机构专用席位买入
   - 游资活跃度

3. **融资融券**
   - 融资余额变化趋势
   - 融券余额异常

4. **大宗交易**
   - 折价/溢价率
   - 交易量异常

5. **限售解禁**
   - 未来 30 天内解禁规模
   - 占流通市值比例

6. **板块轮动**
   - `get_concept_list` — 热门概念板块
   - `get_industry_list` — 行业涨幅排名
   - 资金流入最多的行业/概念

### 第四步：输出投资机会

对每个筛选出的标的提供：

| 维度 | 内容 |
|------|------|
| **标的** | 名称（代码） |
| **行业** | 申万行业分类 |
| **市值** | 当前总市值 |
| **核心逻辑** | 1-2 句话说明投资逻辑 |
| **催化剂** | 近期可能的股价催化事件 |
| **估值** | 当前 PE/PB vs 历史分位 |
| **风险** | 主要风险点 |
| **评分** | 综合吸引力 1-5 星 |

## MCP 工具使用

- **`get_stock_list`** — 获取全部 A 股列表
- **`get_realtime_quote`** — 实时行情（筛选条件）
- **`get_valuation_metrics`** — PE/PB 等估值
- **`get_financial_indicators`** — ROE/增速等
- **`get_industry_list`** — 行业分类
- **`get_concept_list`** — 概念板块
- **`get_market_overview`** — 市场总览

---

## 注意事项

- 筛选结果仅为研究参考，不构成投资建议
- 量化筛选有滞后性，需结合定性判断
- 注意 ST/*ST 公司的特殊风险
- 科创板/创业板/北交所有不同的交易规则和投资者门槛
