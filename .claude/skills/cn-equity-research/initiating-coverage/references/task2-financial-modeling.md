# 任务2：财务建模

## 目标
构建完整的三表预测模型，为估值提供基础数据。

## 数据采集

### 通过 cn-financial-mcp 获取
```
get_income_statement(symbol, period="annual")    → 历史利润表（至少3年）
get_balance_sheet(symbol, period="annual")        → 历史资产负债表
get_cash_flow(symbol, period="annual")            → 历史现金流量表
get_financial_indicators(symbol)                   → 历史财务指标
get_income_statement(symbol, period="quarterly")  → 季度利润表（分析季节性）
```

## 建模步骤

### 步骤 1：历史数据整理

**标准化处理：**
1. 统一单位（万元或百万元）
2. 计算关键比率（毛利率、费用率、周转天数等）
3. 识别非经常性项目并单独列示
4. 标注会计政策变更和追溯调整

**历史数据表：**
```
                    2021A    2022A    2023A    2024A
营业收入             XXX      XXX      XXX      XXX
  YoY               XX%      XX%      XX%      XX%
毛利率              XX.X%    XX.X%    XX.X%    XX.X%
销售费用率           XX.X%    XX.X%    XX.X%    XX.X%
管理费用率           XX.X%    XX.X%    XX.X%    XX.X%
研发费用率           XX.X%    XX.X%    XX.X%    XX.X%
净利率              XX.X%    XX.X%    XX.X%    XX.X%
ROE                 XX.X%    XX.X%    XX.X%    XX.X%
```

### 步骤 2：营收预测

**自上而下法：**
```
行业市场规模 × 市场份额 = 公司营收
```

**自下而上法：**
```
业务线A：销量 × 单价 = 收入A
业务线B：客户数 × ARPU = 收入B
业务线C：产能 × 利用率 × 单价 = 收入C
```

**关键假设：**
- 各业务线增速及驱动因素
- 新产品/新客户贡献时点
- 价格趋势判断
- 季节性模式

### 步骤 3：利润表预测

**驱动关系：**
| 科目 | 驱动方式 |
|------|---------|
| 营业成本 | 毛利率假设 或 成本结构拆分（原材料+人工+制造费用） |
| 销售费用 | 占收入比例（参考历史趋势和扩张计划） |
| 管理费用 | 占收入比例（考虑规模效应） |
| 研发费用 | 占收入比例（考虑研发投入规划） |
| 财务费用 | 有息负债 × 加权利率 |
| 投资收益 | 基于投资资产和历史收益率 |
| 所得税 | 利润总额 × 有效税率 |

### 步骤 4：资产负债表预测

**营运资本驱动：**
```
应收账款 = 营收 × (应收周转天数/365)
存货 = 营业成本 × (存货周转天数/365)
应付账款 = 营业成本 × (应付周转天数/365)
预付款项 = 营收 × 历史占比
预收款项/合同负债 = 营收 × 历史占比
```

**长期资产驱动：**
```
固定资产 = 期初 + CAPEX - 折旧 - 处置
在建工程 = 期初 + 新增投资 - 转固
无形资产 = 期初 + 新增 - 摊销
```

### 步骤 5：现金流量表预测

**间接法构建经营现金流：**
```
经营现金流 = 净利润
  + 折旧摊销
  + 资产减值损失
  - △应收账款
  - △存货
  + △应付账款
  + △预收款/合同负债
  - △预付款项
  ± 其他调整项
```

**投资与筹资现金流：**
```
投资现金流 = -CAPEX - 股权投资 + 投资收回
筹资现金流 = 新增借款 - 偿还借款 - 分红支出 ± 股权融资
```

### 步骤 6：三表勾稽验证

**必须满足：**
1. 资产 = 负债 + 权益（每个预测年份）
2. 期末现金 = 期初现金 + 三项现金流净额
3. 净利润 → 未分配利润的传导正确
4. 少数股东权益随少数股东损益变动

## 模型质量检查

| 检查项 | 标准 |
|--------|------|
| 三表是否平衡 | 误差 < 1元 |
| 关键比率趋势是否合理 | 无异常跳变 |
| 预测期末比率是否趋于稳态 | 增速收敛至行业水平 |
| CAPEX与折旧关系 | 成熟期 CAPEX ≈ 折旧 |
| 经营现金流/净利润 | 长期应 > 0.8 |
| 资产负债率趋势 | 不持续升高或过度下降 |

## 输出

- 完整三表预测（3年历史 + 3年预测）
- 关键假设汇总表
- 敏感性分析（营收增速 × 利润率）
