# 任务5：报告组装

## 目标
将前四个任务的产出整合为一份完整的首次覆盖研究报告。

## 报告结构（30-50页）

### 封面（1页）
```
[券商Logo]

首次覆盖报告

[公司名称]（[股票代码].[交易所]）
[一句话投资概要]

评级：买入 / 增持 / 中性
目标价：XX.XX 元（当前价 XX.XX 元，上行空间 XX%）

行业：[申万二级行业]
日期：YYYY年MM月DD日
分析师：[姓名]
```

### 投资概要（1-2页）
- **核心逻辑**：3-5 个关键投资理由（每个 2-3 句）
- **目标价与估值**：简述估值方法和结果
- **关键财务数据表**：

| 指标 | 2023A | 2024A | 2025E | 2026E | 2027E |
|------|-------|-------|-------|-------|-------|
| 营收(亿) | | | | | |
| 归母净利(亿) | | | | | |
| EPS(元) | | | | | |
| PE | | | | | |
| PB | | | | | |
| ROE | | | | | |
| 股息率 | | | | | |

- **催化剂**：短期（3个月内）和中期（6-12个月）
- **关键风险**：Top 3 风险因素

### 公司概况（3-5页）
- 来自任务1的公司研究
- 公司简介、发展历程
- 业务模式、产品/服务矩阵
- 管理层与治理结构
- 股权结构

### 行业分析（3-5页）
- 行业规模与增长趋势
- 竞争格局（CR5、主要玩家）
- 产业链分析
- 政策环境与监管趋势
- 行业驱动因素与风险

### 竞争优势分析（3-5页）
- 核心竞争壁垒
- 与竞争对手的定量对比
- SWOT 分析
- 波特五力分析

### 财务分析（5-8页）
- 来自任务2的财务建模
- 历史财务回顾
- 盈利能力分析（毛利率、净利率、ROE趋势）
- 资产质量分析（应收、存货、商誉）
- 现金流分析
- 偿债能力分析

### 盈利预测（3-5页）
- 来自任务2的预测模型
- 关键假设说明
- 三表预测（利润表、资产负债表、现金流量表）
- 敏感性分析

### 估值分析（3-5页）
- 来自任务3的估值
- 多方法估值汇总
- 可比公司估值表
- PE Band / PB Band 图表
- 目标价推导

### 投资风险（1-2页）
- 行业风险
- 公司特有风险
- 政策风险
- 估值风险
- 市场风险

### 附录（1-3页）
- 完整财务报表预测
- 可比公司详细数据
- 术语解释
- 免责声明

## 组装规范

### 章节衔接
- 每章开头有 1-2 句过渡语
- 章节之间的数据引用保持一致
- 前后提到的数字必须一致

### 交叉引用检查
- [ ] 投资概要中的财务数据 = 盈利预测中的数据
- [ ] 目标价推导过程 = 估值分析中的结论
- [ ] 风险提示覆盖了正文中提到的所有不确定性
- [ ] 图表数据与文字描述一致

### 格式一致性
- [ ] 所有金额单位统一（亿元或百万元）
- [ ] 所有比率保留位数一致
- [ ] 年份标注格式统一（2025E）
- [ ] 页眉页脚格式统一
- [ ] 字体字号一致

### 数据来源标注
- 公司财报数据 → "来源：公司公告"
- MCP获取数据 → "来源：cn-financial-mcp"
- 行业数据 → "来源：[具体来源名称]"
- 预测数据 → "来源：分析师预测"

## 质量评估标准

| 维度 | 评分标准 |
|------|---------|
| 逻辑连贯性 | 投资逻辑从头到尾自洽，没有自相矛盾 |
| 数据准确性 | 所有数字可追溯、可验证 |
| 分析深度 | 超越数据罗列，有独立见解和因果分析 |
| 覆盖完整性 | 关键风险和机会都有讨论 |
| 可读性 | 结构清晰，关键信息突出 |
| 专业性 | 术语使用准确，格式符合券商研报标准 |

## 最终检查清单

- [ ] 报告总页数在 30-50 页范围内
- [ ] 投资评级和目标价在封面、概要、估值章节三处一致
- [ ] 所有图表编号和引用正确
- [ ] 非经常性损益已单独标注
- [ ] 历史数据 vs 预测数据有明确区分
- [ ] 免责声明包含在内
- [ ] 拼写和语法检查通过
