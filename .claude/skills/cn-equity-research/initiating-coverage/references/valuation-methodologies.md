# A股估值方法论参考

## 估值方法选择指南

### 按行业选择

| 行业大类 | 首选方法 | 辅助方法 | 不适用方法 |
|---------|---------|---------|-----------|
| 白酒/高端消费 | PE + DCF | PEG | PB（轻资产） |
| 银行 | PB-ROE + DDM | PE | PS、EV/EBITDA |
| 保险 | 内含价值 (EV) | PB-ROE | PE（利润波动大） |
| 证券 | PB-ROE | PE | DCF（业绩周期性强） |
| 医药（创新药） | DCF/管线估值 | PS | PE（尚未盈利） |
| 医药（仿制药） | PE + PB | DCF | PS |
| 科技/SaaS | PS + DCF | PE（远期） | PB |
| 半导体 | PE/PB 周期择时 | EV/EBITDA | DDM |
| 新能源（锂电/光伏） | PE + PEG | EV/EBITDA | PB |
| 房地产 | NAV + PB | PE | DCF |
| 基建/建筑 | PE + PB | EV/EBITDA | DCF |
| 公用事业 | DDM + PE | DCF | PS |
| 传媒/游戏 | PE + PS | DCF | PB |
| 制造业 | PE + EV/EBITDA | PB | PS |

### 按公司阶段选择

| 阶段 | 特征 | 适用方法 |
|------|------|---------|
| 初创/亏损 | 无利润 | PS、DCF（收入驱动）、可比交易 |
| 高速成长 | 利润增速 > 30% | PEG、PS、DCF |
| 稳定成长 | 利润增速 10-30% | PE、DCF、EV/EBITDA |
| 成熟稳定 | 利润增速 < 10% | PE、DDM、EV/EBITDA |
| 周期底部 | 利润极低或亏损 | PB、EV/EBITDA、重置成本 |

## PE 估值详解

### 估值步骤
1. 确定预测 EPS（通常用 T+1 年或 T+2 年）
2. 确定合理 PE 倍数
3. 目标价 = EPS × PE

### 合理 PE 的确定

**方法A：可比公司法**
```
目标 PE = 可比公司 PE 中位数 × (1 + 增速溢价/折价)
```
- 增速高于同行 → 溢价
- ROE 高于同行 → 溢价
- 公司治理差 → 折价
- 流动性差（小盘）→ 折价

**方法B：历史中枢法**
```
目标 PE = 公司3-5年 PE-TTM 均值或中位数
```
- 适用于业务模式稳定、增速变化不大的公司
- 需排除异常年份（如2020年疫情、行业周期底部）

**方法C：PEG法**
```
合理 PE = 预期净利润增速(%) × PEG
PEG = 1.0 为基准（增速30% → PE 30x）
```
- 高确定性公司 PEG 可给 1.0-1.5
- 低确定性/周期性公司 PEG 应给 0.5-0.8

### A股 PE 溢价/折价因素

| 因素 | 溢价/折价 | 幅度 |
|------|---------|------|
| 行业龙头 | 溢价 | +10-30% |
| 国产替代受益 | 溢价 | +10-20% |
| 沪深300成分 | 溢价 | +5-10% |
| 北向资金重仓 | 溢价 | +5-15% |
| 大股东质押高 | 折价 | -10-20% |
| 商誉占比高 | 折价 | -10-20% |
| 关联交易频繁 | 折价 | -5-15% |
| 审计非标 | 折价 | -20-50% |

## PB-ROE 估值详解

### 理论基础
```
PB = (ROE - g) / (Ke - g)
```
推论：
- ROE > Ke → PB > 1（值得溢价）
- ROE = Ke → PB = 1（合理定价）
- ROE < Ke → PB < 1（应该折价）

### 银行 PB 估值参考

| ROE 水平 | 资产质量 | 合理 PB |
|---------|---------|---------|
| > 15% | 优秀（不良率 < 1%） | 1.2-1.8x |
| 12-15% | 良好（不良率 1-1.5%） | 0.8-1.2x |
| 8-12% | 一般（不良率 1.5-2%） | 0.5-0.8x |
| < 8% | 较差（不良率 > 2%） | < 0.5x |

## DCF 估值要点（A股特色）

### WACC 参数

| 参数 | 数据源 | A股特点 |
|------|--------|--------|
| Rf（无风险利率） | 10年期国债收益率 | 通常 2.0-3.5% |
| ERP（股权风险溢价） | 历史法/隐含法 | A股通常 5-8%，高于美股 |
| Beta | 对沪深300/中证800回归 | 2年周数据 |
| Kd（债务成本） | 公司实际融资利率 | 参考LPR + 信用利差 |
| 税率 | 公司有效税率 | 高新15%，一般25% |

### 终值处理
- 永续增长率 g 不超过 3-4%（中国GDP长期增速预期）
- 退出倍数法参考成熟市场同行业估值
- 终值占比建议 < 75%

## 分部估值法（SOTP）

适用于多元化经营的公司：

```
公司总价值 = Σ (各业务板块价值)
各板块价值 = 板块利润/收入 × 对应估值倍数（参考纯正同行）
```

### 示例
```
消费品业务：净利润 50亿 × PE 30x = 1,500亿
金融业务：净资产 200亿 × PB 1.5x = 300亿
科技业务：营收 100亿 × PS 5x = 500亿
────────────────────────────────────────
总价值 = 2,300亿
集团折价（10-20%）→ 目标市值 1,840-2,070亿
```

## 跨市场估值对比

A股 vs H股同一公司：
```
AH 溢价率 = (A股价格 / H股价格 × 汇率 - 1) × 100%
```

当前 AH 溢价均值约 30-40%，影响因素：
- 资金面差异（A股散户多、流动性充裕）
- 投资者结构差异
- 互联互通额度限制
- 分红税差异
