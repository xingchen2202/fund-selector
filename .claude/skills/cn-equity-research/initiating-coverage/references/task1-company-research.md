# 任务1：公司研究

## 目标
全面了解目标公司的业务模式、竞争地位、管理层和发展历程，为后续财务建模和估值奠定基础。

## 数据采集

### 通过 cn-financial-mcp 获取
```
search_stock(keyword)           → 确认公司代码
get_company_info(symbol)        → 公司基本信息（行业、主营、注册地等）
get_stock_news(symbol)          → 近期新闻和公告
get_competitors(symbol)         → 竞争对手列表
get_industry_stocks(industry)   → 行业内所有上市公司
```

### 补充信息来源
- **招股说明书**（IPO文件）：最详细的业务描述，巨潮资讯网下载
- **年报 MD&A**：管理层对经营状况的讨论
- **调研纪要**：券商/基金公司调研纪要（需合规获取）
- **行业报告**：券商研究所行业深度报告

## 分析框架

### 1. 业务模式画布

| 维度 | 分析内容 |
|------|---------|
| 价值主张 | 核心产品/服务解决什么问题 |
| 客户群体 | B端/C端/G端，大客户集中度 |
| 收入模式 | 产品销售/服务费/平台抽成/订阅 |
| 关键资源 | 技术专利/品牌/渠道/牌照 |
| 成本结构 | 固定成本 vs 变动成本比例 |
| 竞争壁垒 | 规模效应/网络效应/转换成本/品牌/技术 |

### 2. 发展历程梳理

按时间线整理关键事件：
- 成立与上市时间
- 重大并购与资产重组
- 产品线扩展
- 区域扩张
- 管理层变更
- 重大政策影响

### 3. 管理层评估

| 维度 | 关注点 |
|------|--------|
| 实际控制人 | 持股比例、一致行动人、质押情况 |
| 管理团队 | 行业经验、任职年限、股权激励绑定 |
| 治理结构 | 独立董事比例、审计委员会、关联交易管控 |
| 过往记录 | 是否有信批违规、业绩承诺违约 |

### 4. 行业定位

- 所属申万行业分类（一级→二级→三级）
- 行业生命周期阶段（导入期/成长期/成熟期/衰退期）
- 市场份额及排名
- 行业集中度趋势（CR3/CR5/CR10）
- 产业链位置（上游/中游/下游）

## 输出要求

完成公司研究后，应能回答以下核心问题：
1. 公司靠什么赚钱？（商业模式）
2. 为什么能持续赚钱？（竞争壁垒）
3. 未来增长来自哪里？（增长驱动力）
4. 最大的风险是什么？（核心风险）
5. 管理层是否值得信任？（治理与激励）

## 写作要点

- 用自己的语言概括，而非照搬年报原文
- 对每个业务线给出简洁的盈利驱动公式：收入 = 量 × 价
- 关键数据须附来源标注
- 篇幅控制：首次覆盖报告中，公司研究部分 8-12 页
