# 任务3：估值分析

## 目标
使用多种估值方法对目标公司进行合理定价，得出目标价和投资评级。

## 数据采集

### 通过 cn-financial-mcp 获取
```
get_valuation_metrics(symbol)                     → 当前估值倍数
get_historical_price(symbol, period="monthly")    → 历史股价（计算历史估值区间）
get_competitors(symbol)                            → 可比公司
get_industry_stocks(industry)                      → 行业公司列表
# 对每家可比公司获取：
get_valuation_metrics(comp_symbol)                → 可比公司当前估值
get_financial_indicators(comp_symbol)             → 可比公司财务指标
```

## 估值方法体系

### 方法一：PE 估值（最常用）

**适用场景**：盈利稳定、可预测性强的公司

```
目标价 = 预测 EPS × 目标 PE 倍数
```

**目标 PE 确定方法：**
1. 可比公司当前 PE 中位数
2. 公司历史 PE 均值/中位数（3-5年）
3. PEG = 1 对应的合理 PE（PE = 净利润增速%）
4. 行业平均 PE

**A股 PE 参考区间：**
| 行业 | PE-TTM 中位数范围 | 说明 |
|------|------------------|------|
| 白酒 | 25-40x | 高确定性溢价 |
| 银行 | 5-8x | 低估值常态 |
| 医药 | 25-50x | 创新药估值弹性大 |
| 电子/半导体 | 30-60x | 周期+成长 |
| 新能源 | 20-50x | 成长预期驱动 |
| 消费品 | 20-35x | 稳定增长 |
| 地产 | 5-10x | 行业下行周期 |

### 方法二：PB-ROE 估值

**适用场景**：银行、保险、证券等金融行业；重资产行业

```
目标价 = 每股净资产 × 目标 PB 倍数
目标 PB ≈ (ROE - g) / (Ke - g)    （戈登增长模型推导）
```

**PB-ROE 框架：**
| ROE | 合理 PB |
|-----|---------|
| < 8% | < 1.0x |
| 8-12% | 1.0-1.5x |
| 12-18% | 1.5-3.0x |
| > 18% | 3.0x+ |

### 方法三：DCF 估值

**适用场景**：现金流可预测、增长确定性高的公司

详见 `cn-financial-analysis/skills/dcf-model/` 的完整 DCF 模型技能。

**简化框架：**
```
企业价值 (EV) = Σ [FCFFt / (1+WACC)^t] + TV / (1+WACC)^n
股权价值 = EV - 净负债 - 少数股东权益
每股价值 = 股权价值 / 总股本
```

### 方法四：PS 估值

**适用场景**：亏损或微利的高成长公司

```
目标价 = 预测每股营收 × 目标 PS 倍数
```

### 方法五：EV/EBITDA

**适用场景**：资本结构差异大的行业对比、跨国比较

```
目标 EV = 预测 EBITDA × 目标 EV/EBITDA
目标价 = (目标 EV - 净负债 - 少数股东权益) / 总股本
```

### 方法六：NAV（净资产价值法）

**适用场景**：房地产、资源类公司

```
NAV = Σ 各项目/资产重估价值 - 净负债
每股 NAV = NAV / 总股本
目标价 = 每股 NAV × NAV 折溢价率
```

### 方法七：DDM（股利折现模型）

**适用场景**：高分红、股利政策稳定的公司（银行、公用事业）

```
每股价值 = D1 / (Ke - g)
D1 = 预测每股股利
g = 股利永续增长率
```

### 权重分配原则
- 盈利稳定型：PE 50% + DCF 30% + EV/EBITDA 20%
- 高成长型：DCF 40% + PE（远期）30% + PS 30%
- 金融类：PB-ROE 50% + DDM 30% + PE 20%
- 重资产/地产：NAV 50% + PB 30% + PE 20%

### 投资评级标准

| 评级 | 上行空间 | 说明 |
|------|---------|------|
| 买入 | > 15% | 显著低估 |
| 增持 | 5-15% | 小幅低估 |
| 中性 | -5% ~ +5% | 合理估值 |
| 减持 | < -5% | 高估 |
