## 图表设计规范

### 颜色方案
```
主色:    #1A56DB (蓝)    - 主要数据系列
辅色1:   #E74C3C (红)    - 对比/负面数据
辅色2:   #2ECC71 (绿)    - 正面/增长数据
辅色3:   #F39C12 (橙)    - 次要数据系列
辅色4:   #9B59B6 (紫)    - 第三数据系列
背景:    #FFFFFF (白)    - 图表背景
网格线:  #E5E7EB (浅灰)  - 辅助线
```

### 图表元素
- **标题**：简洁明了，含时间范围（如 "营收及净利润趋势 (2021-2027E)"）
- **图例**：放在图表上方或右侧
- **数据标签**：关键转折点标注数值
- **Y 轴**：金额用"亿元"或"百万元"，比率用"%"
- **X 轴**：年份用 "2023A / 2024E / 2025E" 格式
- **来源**：图表下方注明 "数据来源：公司公告, cn-financial-mcp"
- **预测分隔**：历史与预测之间用虚线分隔

### 文字标注规范
- 字体：衬线体或无衬线体均可（中文推荐微软雅黑/思源黑体）
- 标题字号：14-16pt
- 轴标签：10-12pt
- 数据标签：9-10pt

## 图表生成方式

### 方式一：Markdown 表格 + 描述
适用于文本报告，用 Markdown 表格呈现数据，辅以文字描述趋势。

### 方式二：Python matplotlib/plotly
如果用户环境支持 Python：
```python
import matplotlib.pyplot as plt
import matplotlib
matplotlib.rcParams['font.sans-serif'] = ['SimHei']  # 中文显示
matplotlib.rcParams['axes.unicode_minus'] = False

# 示例：营收趋势图
years = ['2022A', '2023A', '2024A', '2025E', '2026E', '2027E']
revenue = [100, 120, 145, 170, 195, 220]
net_profit = [30, 38, 46, 55, 64, 73]

fig, ax1 = plt.subplots(figsize=(10, 6))
ax1.bar(years, revenue, color='#1A56DB', alpha=0.7, label='营业收入')
ax2 = ax1.twinx()
ax2.plot(years, net_profit, color='#E74C3C', marker='o', label='归母净利润')

ax1.set_ylabel('营业收入 (亿元)')
ax2.set_ylabel('归母净利润 (亿元)')
plt.title('营收及净利润趋势')
plt.axvline(x=2.5, color='gray', linestyle='--', alpha=0.5)  # 历史|预测分隔
```

## PE Band 图表说明

PE Band 是A股研报中最常用的估值图表：

```
股价
  ^
  |    ╱─── 40x PE
  |  ╱───── 30x PE
  | ╱────── 25x PE （中枢）
  |╱─────── 20x PE
  ╱──────── 15x PE
  |
  +──────────────────→ 时间
  2020   2021   2022   2023   2024   2025
```

计算方式：
```
PE Band 线 = 滚动EPS × 对应PE倍数
```
