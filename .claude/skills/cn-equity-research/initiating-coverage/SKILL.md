# A 股首次覆盖报告（深度研报）

创建机构级 **首次覆盖报告**，遵循国内头部券商研究所和外资行中国研究团队标准。

**核心特征：**
- **篇幅**：30-50 页
- **字数**：10,000-20,000 字
- **表格**：10-15 张
- **图表**：20-30 张
- **周期**：3-6 周
- **受众**：机构投资者（公募/私募/保险/QFII）
- **核心产出**：投资评级 + 目标价 + 完整估值

## 可用 MCP 工具

- **`search_stock`** — 搜索公司和同行
- **`get_company_info`** — 公司基本信息、主营业务
- **`get_income_statement`** — 利润表（多年）
- **`get_balance_sheet`** — 资产负债表（多年）
- **`get_cash_flow`** — 现金流量表（多年）
- **`get_financial_indicators`** — ROE、毛利率等指标
- **`get_valuation_metrics`** — PE、PB 等估值
- **`get_historical_price`** — 历史股价
- **`get_realtime_quote`** — 最新行情
- **`get_industry_stocks`** — 同行业公司
- **`get_stock_news`** — 相关新闻

## 报告结构

### 一、投资摘要（2-3 页）

1. **核心投资逻辑**（3-5 条，每条 2-3 句）
2. **投资评级与目标价**
   - 评级：买入 / 增持 / 中性 / 减持
   - 目标价及估值方法
   - 当前股价及上涨空间
3. **关键财务预测摘要**

| 指标 | 2023A | 2024A | 2025E | 2026E | 2027E |
|------|-------|-------|-------|-------|-------|
| 营业收入（亿元） | | | | | |
| 归母净利润（亿元） | | | | | |
| EPS（元） | | | | | |
| PE-TTM | | | | | |
| PB | | | | | |
| ROE | | | | | |

### 二、公司概况（3-5 页）

1. **公司简介** — 历史沿革、股权结构、管理层
2. **业务概述** — 主营业务、产品线、商业模式
3. **收入结构** — 按产品/地区/客户拆分
4. **核心竞争力** — 技术、品牌、渠道、规模

### 三、行业分析（5-8 页）

1. **行业空间** — TAM/SAM/SOM 测算
2. **行业增速** — 历史和预测
3. **竞争格局** — CR3/CR5、主要玩家
4. **产业链** — 上下游关系和议价能力
5. **产业政策** — 支持/限制性政策梳理
6. **行业趋势** — 技术变革、消费升级、国产替代等

### 四、业务深度分析（5-8 页）

按业务线逐一分析：
1. 业务描述和定位
2. 市场份额和竞争地位
3. 增长驱动力
4. 盈利模式和利润率
5. 风险因素

### 五、财务分析（5-8 页）

1. **盈利能力**
   - 毛利率、净利率趋势及驱动
   - 费用率分析（三费 + 研发）
   - 扣非净利 vs 归母净利

2. **资产质量**
   - 资产负债率和有息负债
   - 应收/存货周转
   - 商誉和减值风险

3. **现金流质量**
   - 经营现金流/净利润比
   - 自由现金流
   - CapEx 强度

4. **ROE 杜邦分解**
   - ROE = 净利率 × 资产周转率 × 权益乘数

### 六、盈利预测（3-5 页）

1. **预测假设**
   - 收入端：量 × 价，或增速假设
   - 成本端：毛利率假设
   - 费用端：费用率假设
2. **三表预测**（参考 cn-3-statements 技能）
3. **预测敏感性**

### 七、估值分析（3-5 页）

**A 股常用估值方法：**

1. **PE 估值法**
   - 选取可比公司 PE 中位数
   - 给予合理 PE 倍数（考虑增速差异）
   - 目标价 = 预测 EPS × 目标 PE

2. **PB-ROE 估值法**
   - PB = ROE × PE
   - 适用于金融、地产等重资产行业

3. **DCF 估值法**
   - 参考 cn-dcf-model 技能
   - 人民币 WACC

4. **PEG 估值法**
   - PEG = PE / 盈利增速
   - PEG < 1 被认为低估

5. **可比公司法**
   - 参考 cn-comps-analysis 技能

**估值汇总表：**
| 方法 | 假设 | 目标价（元） |
|------|------|-------------|
| PE | 25x 2026E EPS | |
| PB-ROE | 3.5x 2025E BVPS | |
| DCF | WACC=10%, g=3% | |
| 平均 | | |

### 八、风险提示（1 页）

列示 5-8 条主要风险：
- 行业竞争加剧
- 原材料价格波动
- 政策变化风险
- 宏观经济下行
- 技术路线变更
- 大客户集中度
- 汇率波动（如有出口）
