# 首次覆盖报告质量检查清单

## 一、结构完整性

- [ ] 封面包含：公司名称、代码、评级、目标价、行业、日期
- [ ] 投资概要不超过 2 页
- [ ] 公司概况章节完整
- [ ] 行业分析有深度
- [ ] 竞争优势分析有定量支撑
- [ ] 财务分析覆盖盈利、资产、现金流
- [ ] 盈利预测包含三表
- [ ] 估值分析使用多种方法
- [ ] 风险提示覆盖全面
- [ ] 附录包含完整数据
- [ ] 总页数在 30-50 页范围内

## 二、投资逻辑一致性

- [ ] 投资概要中的核心逻辑在正文中有充分论据
- [ ] 评级（买入/增持/中性/减持）与上行空间匹配
- [ ] 目标价推导过程清晰可追溯
- [ ] 风险提示与投资逻辑的隐含假设对应
- [ ] 催化剂有明确的时间窗口

## 三、数据准确性

- [ ] 历史财务数据与公司公告一致
- [ ] 同一数字在报告不同位置保持一致
- [ ] 计算的比率和百分比数学正确
- [ ] 三表勾稽（资产=负债+权益）
- [ ] 预测年份的现金余额与资产负债表一致
- [ ] EPS = 归母净利润 / 总股本，股本数正确
- [ ] PE = 股价 / EPS，数据时点一致

## 四、分析深度

- [ ] 营收拆分到业务线/产品/区域层面
- [ ] 成本分析包含原材料、人工等具体项目
- [ ] 不仅有"是什么"，还有"为什么"的因果分析
- [ ] 与竞争对手的定量比较（而非仅定性）
- [ ] 对管理层指引/市场一致预期有独立判断
- [ ] 敏感性分析覆盖关键变量

## 五、A股特色要素

- [ ] 扣非净利润已单独分析
- [ ] 非经常性损益项目已识别
- [ ] 关联交易情况已提及
- [ ] 大股东持股/质押情况已说明
- [ ] 限售股解禁时间表已提及
- [ ] 股权激励计划（如有）已分析
- [ ] 政策风险已评估（行业监管、产业政策）
- [ ] 北向资金持仓变动已关注
- [ ] 分红方案/分红率已分析

## 六、估值合理性

- [ ] 使用至少 2 种估值方法
- [ ] 可比公司选择有合理依据
- [ ] PE 倍数在行业合理区间
- [ ] DCF 的 WACC 在合理范围（6-12%）
- [ ] 永续增长率不超过 GDP 长期增速
- [ ] 终值占 EV 比例 < 75%
- [ ] 估值隐含的增长假设与预测一致

## 七、风险评估

- [ ] 至少列出 5 个风险因素
- [ ] 风险按影响程度排序
- [ ] 每个风险有具体描述（非泛泛而谈）
- [ ] 包含行业系统性风险
- [ ] 包含公司特有风险
- [ ] 包含政策/监管风险
- [ ] 估值风险已提及

## 八、格式与规范

- [ ] 数字单位统一（亿元/百万元）
- [ ] 百分比保留一位小数
- [ ] 历史数据标注"A"，预测数据标注"E"
- [ ] 图表有标题、来源标注
- [ ] 关键术语一致使用
- [ ] 无错别字和语法错误
- [ ] 数据来源标注完整

## 评分标准

| 得分 | 等级 | 说明 |
|------|------|------|
| 90-100% | A | 机构级质量，可直接发布 |
| 80-89% | B | 良好，需小幅修改 |
| 70-79% | C | 合格，需补充部分内容 |
| < 70% | D | 不合格，需大幅修改 |
