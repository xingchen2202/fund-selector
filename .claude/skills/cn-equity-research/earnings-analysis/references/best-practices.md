# A股财报分析最佳实践

## 核心原则

### 1. 先看扣非，再看净利
- A股公司非经常性损益波动大（政府补助、投资收益、资产处置）
- **扣非净利润**才是衡量主业经营能力的核心指标
- 非经常性损益占比超过净利润 20% 时需重点标注

### 2. 经营现金流 vs 净利润
- 持续关注经营现金流/净利润比值（健康区间 > 0.8）
- 净利润高但经营现金流差 → 可能存在应收账款激增、收入确认激进
- 典型行业差异：
  - 白酒/消费：现金流通常 > 净利润（预收款模式）
  - 工程/地产：现金流可能大幅落后净利润（项目周期）
  - 互联网/软件：SaaS 模式预收年费，现金流优于利润

### 3. 季度拆分分析
- A股季报不提供完整三表，需手动拆分：
  - Q2单季 = H1累计 - Q1
  - Q3单季 = 前三季度累计 - H1
  - Q4单季 = 全年 - 前三季度累计
- 注意Q4异常：年末集中确认收入、资产减值计提、费用集中处理

### 4. 关注合并报表 vs 母公司报表
- 合并报表反映集团整体，母公司报表反映控股公司本体
- 投资收益差异 → 子公司利润贡献
- 应收账款差异 → 关联方往来

## 财报阅读顺序

### 第一步：审计意见（年报）
- **标准无保留** → 正常
- **带强调事项段** → 需关注特定风险
- **保留意见/否定意见/无法表示意见** → 重大红旗 🚩

### 第二步：关键财务数据
1. 营收和归母净利润及增速
2. 扣非净利润及增速
3. 经营活动现金流量净额
4. 毛利率、净利率变化趋势
5. 加权 ROE

### 第三步：管理层讨论与分析 (MD&A)
- 行业环境判断
- 核心竞争力描述变化
- 未来经营计划的具体程度
- 风险提示新增内容

### 第四步：深度科目分析
- 应收账款账龄及坏账准备
- 存货构成（原材料/在产品/产成品）
- 在建工程进展及转固时点
- 商誉减值风险
- 关联交易明细

## A股特色分析要点

### 政府补助
- 计入"其他收益"或冲减成本/费用
- 区分与资产相关 vs 与收益相关
- 关注补助的可持续性（一次性 vs 常规性）

### 股权激励
- 关注限制性股票/股票期权的摊销费用
- 行权价格与当前股价的关系
- 解锁条件（业绩考核目标）反映管理层信心

### 研发资本化
- 资本化比例高 → 当期利润被美化
- 比较同行业资本化政策差异
- 关注资本化研发的后续摊销对利润的影响

### 资产减值信号
- 应收账款坏账准备计提比例变化
- 存货跌价准备
- 商誉减值测试（关注并购标的业绩承诺完成情况）
- 长期资产减值（固定资产、无形资产）

## 关键红旗清单 🚩

| 红旗 | 说明 |
|------|------|
| 营收增长但现金流下降 | 可能收入确认激进或回款恶化 |
| 应收账款增速 >> 营收增速 | 信用政策放松，坏账风险上升 |
| 存货异常增长 | 产品滞销或囤货，关注跌价风险 |
| 经营现金流持续为负 | 主业造血能力不足，依赖融资维持 |
| 频繁变更会计政策/估计 | 可能在调节利润 |
| 关联交易占比过高 | 利润输送或转移风险 |
| 商誉占净资产比例 > 30% | 大额减值风险 |
| 审计费用大幅上升 | 可能反映审计发现问题 |
| 非标审计意见 | 重大财务风险信号 |
| 大股东质押比例 > 50% | 流动性风险 + 公司治理隐忧 |
