# A 股行业综述

创建全面的 **行业综述报告**，基于申万行业分类体系，覆盖行业全景分析。

**核心特征：**
- **篇幅**：15-25 页
- **字数**：6,000-10,000 字
- **表格**：5-8 张
- **图表**：10-15 张
- **更新频率**：季度或半年度

## 可用 MCP 工具

- **`get_industry_list`** — 获取申万行业列表和涨跌幅
- **`get_industry_stocks`** — 获取行业成分股
- **`get_concept_list`** — 概念板块（产业链细分）
- **`get_financial_indicators`** — 行业内各公司财务指标
- **`get_valuation_metrics`** — 行业估值水平
- **`get_realtime_quote`** — 成分股行情
- **`get_stock_news`** — 行业相关新闻

## 报告结构

### 一、行业概况（2-3 页）

1. **行业定义与范围**
   - 申万行业分类（一级 → 二级 → 三级）
   - 行业涵盖的细分领域
   - 与其他分类体系的对应关系

2. **行业规模与增速**
   - 总体市场规模（TAM）
   - 近 5 年增速趋势
   - 未来 3-5 年增速预测
   - 增长驱动因素

3. **行业生命周期定位**
   - 导入期 / 成长期 / 成熟期 / 衰退期

### 二、产业链分析（2-3 页）

1. **上游** — 原材料/零部件/技术供给
2. **中游** — 制造/加工/服务
3. **下游** — 应用场景/终端需求

每个环节分析：
- 主要参与者（上市公司优先）
- 价值分配比例
- 议价能力
- 供需关系

### 三、竞争格局（3-4 页）

1. **行业集中度**
   - CR3 / CR5 / CR10 及趋势
   - HHI 指数

2. **主要竞争者**（使用 MCP 获取行业成分股）

| 公司 | 代码 | 市值 | 营收 | 市占率 | 毛利率 | ROE |
|------|------|------|------|--------|--------|-----|
| | | | | | | |

3. **竞争维度**
   - 价格竞争 vs 差异化竞争
   - 技术领先 vs 规模领先
   - 品牌壁垒 vs 渠道壁垒

4. **格局演变趋势**
   - 集中度提升 / 分散
   - 国产替代进展
   - 行业整合（并购）

### 四、产业政策（2-3 页）

1. **国家级政策**
   - "十四五"规划相关内容
   - 产业发展指导目录
   - 财税补贴政策

2. **监管政策**
   - 准入条件
   - 环保标准
   - 安全监管

3. **地方政策**
   - 产业集群
   - 区域扶持

4. **政策影响评估**
   - 对行业空间的影响
   - 对竞争格局的影响
   - 对盈利能力的影响

### 五、景气度跟踪（2-3 页）

**高频指标监控：**

| 指标 | 频率 | 当前值 | 趋势 | 说明 |
|------|------|--------|------|------|
| 行业产量/销量 | 月度 | | | |
| 产品价格 | 周度 | | | |
| 库存水平 | 月度 | | | |
| 产能利用率 | 季度 | | | |
| 新订单/在手订单 | 月度 | | | |
| 行业PMI | 月度 | | | |

**景气度综合判断：**
- 🟢 上行期 / 🟡 平稳期 / 🔴 下行期

### 六、估值分析（2-3 页）

1. **行业估值水平**
   - PE-TTM 中位数 / 均值
   - 当前估值 vs 历史分位（近 3 年 / 5 年）
   - PB 中位数

2. **估值横向比较**
   - 与其他行业对比
   - A/H 股估值差

3. **重点公司估值**

| 公司 | PE-TTM | PB | PS | 历史分位 |
|------|--------|----|----|---------|
| | | | | |

### 七、投资建议（1-2 页）

1. **行业评级** — 超配 / 标配 / 低配
2. **推荐标的**（3-5 只）
   - 名称（代码）
   - 推荐逻辑（1-2 句）
   - 目标估值
3. **风险提示** — 5-8 条
