## 一、核心理念

> **不追 AI 巨头（金枪鱼大腹），寻找被忽视的"瓶颈节点"（紫苏叶）。**

### 投资哲学

| 比喻 | 含义 | 投资逻辑 |
|------|------|---------|
| **金枪鱼大腹** | 人人追捧的 AI 巨头（英伟达/微软/谷歌）| 价值已充分定价，无超额收益 |
| **紫苏叶** | 微型市值、被忽视、但战略稀缺的小公司 | 一旦被发现，弹性巨大 |

### 三层架构

| 层级 | 名称 | 内容 |
|------|------|------|
| **战略核心** | 瓶颈点投资法 | 寻找产业链中不可替代的关键节点 |
| **战术引擎** | 人机协同研究 + 贝叶斯更新 | AI 辅助研究，动态修正判断 |
| **执行筛子** | 五因子模型 | 量化筛选标准 |

## 二、五因子瓶颈评分模型

### 评分标准

| # | 因子 | 标准 | 分值 | 说明 |
|---|------|------|------|------|
| 1 | 细分市占率 | 行业前3 | 3分 | 在细分领域有主导地位 |
| 2 | 毛利率 | > 30% | 3分 | 有定价权，盈利能力强 |
| 3 | 机构持仓 | < 10% | 3分 | 被市场忽视，有发现空间 |
| 4 | 技术壁垒 | 难以替代 | 3分 | 有专利/认证/技术门槛 |
| 5 | 产能约束 | 供给弹性低 | 3分 | 产能紧张，新进入者难 |

### 评级标准

| 总分 | 评级 | 投资建议 |
|------|------|---------|
| **≥ 12/15** | 紫苏叶级（战略稀缺）| **强烈推荐** |
| **9-11/15** | 潜在瓶颈 | **关注** |
| **< 9/15** | 普通持仓 | **观望** |

## 三、产业链瓶颈分析框架

### 分析步骤

1. **选定主题**：确定热门概念（如 AI算力、新能源、半导体）
2. **构建产业链图谱**：上游 → 中游 → 下游
3. **发现上市公司**：各环节的代表公司
4. **五因子评分**：对每只股票评分
5. **筛选瓶颈节点**：评分 ≥ 12 的为紫苏叶级
6. **关联基金发现**：找重仓瓶颈节点的基金
7. **输出报告**：瓶颈地图 + 推荐组合

### 产业链通用结构

```
上游（原材料/设备）
    │
    ▼
中游（制造/组装）
    │
    ▼
下游（应用/服务）
```

## 四、输出格式

### 个股评分输出

```
[紫苏叶评分] XX 股份（XXXXXX）— XX/15
  ✓ 细分市占率第X
  ✓ 毛利率 XX%
  ✓ 机构持仓 XX%
  ✓ 技术壁垒：...
  ✓ 产能约束：...
  评级：紫苏叶级 / 潜在瓶颈 / 普通持仓
```

### 基金紫苏叶指数

```
紫苏叶指数 = 重仓股平均紫苏叶得分 × 持仓集中度

评级：
  ≥ 75：优秀（底层多为瓶颈节点）
  50-74：中等
  < 50：偏低
```

## 五、注意事项

- 持仓数据来自基金季报，**滞后一个季度**
- 产业链图谱为参考，实际数据以 MCP 实时查询为准
- 五因子中"技术壁垒"和"产能约束"为定性判断，需结合行业研究
- 始终附带免责声明
