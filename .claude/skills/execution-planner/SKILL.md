# 交易执行计划

不做单次梭哈预测，而是根据可用资金和市场环境，生成具体的执行计划。

## 触发条件

- "怎么买" / "建仓计划" / "定投方案"
- "XX 基金怎么操作" / "准备 10 万买基金"
- "网格交易" / "金字塔加仓"
- "现在可以加仓吗" / "跌了要不要补"

## 工作流

### 第一步：确认参数

向用户确认以下信息：

1. **标的基金**：基金代码
2. **可用资金**：总金额（元/万元）
3. **策略类型**：
   - DCA 定投（适合震荡市、不确定方向）
   - 金字塔建仓（适合看跌分批买入）
   - 网格交易（适合震荡区间明确的标的）
4. **时间窗口**：计划执行多长时间

### 第二步：获取市场数据

调用以下工具获取决策所需数据：

1. `get_fund_nav_history` — 近 3 个月净值，判断近期趋势
2. `get_fund_info` — 确认基金类型、费率
3. 如是指数基金，额外调用 cn-financial 的 `get_valuation_metrics` 获取指数 PE 分位

### 第三步：生成执行计划

#### 策略 A：DCA 定投

适用场景：方向不明、不想择时、长期看好

```
📋 定投执行计划 — {基金名称}（{代码}）
━━━━━━━━━━━━━━━━━━━━━━━
总资金：{XX} 万元
策略：定期定额投资
执行周期：{N} 个月

┌─────────┬────────┬─────────────────┐
│  月份   │  金额  │  触发条件        │
├─────────┼────────┼─────────────────┤
│ 第1个月 │ ¥{X}   │ 估值<40%分位定投 │
│ 第2个月 │ ¥{X}   │ 估值<40%分位定投 │
│ 第3个月 │ ¥{X}   │ 估值<40%分位定投 │
│ ...     │ ...    │ ...              │
│ 第{N}个月│ ¥{X}   │ 到期或止盈       │
└─────────┴────────┴─────────────────┘

💰 单期金额 = 总资金 / {N} = ¥{X}
📌 加码条件：估值跌至 30% 分位以下，单期金额 ×1.5
📌 止盈条件：浮盈达 20%，暂停定投，锁定一半
📌 止损条件：浮亏达 15%，暂停并复盘

⚠️ 定投不是万能的，在长期阴跌市场中
   需要配合估值判断灵活调整。
```

#### 策略 B：金字塔建仓

适用场景：看好中长期，但短期可能继续跌

```
📋 金字塔建仓计划 — {基金名称}（{代码}）
━━━━━━━━━━━━━━━━━━━━━━━
总资金：{XX} 万元
当前净值：{净值}

┌────────┬────────┬──────────┬─────────────────┐
│  阶段  │  比例  │   金额   │  触发条件        │
├────────┼────────┼──────────┼─────────────────┤
│ 底仓   │  30%   │ ¥{X}     │ 现价直接买入     │
│ 加仓 1 │  30%   │ ¥{X}     │ 净值跌 5%       │
│ 加仓 2 │  20%   │ ¥{X}     │ 净值跌 10%      │
│ 加仓 3 │  20%   │ ¥{X}     │ 净值跌 15%      │
└────────┴────────┴──────────┴─────────────────┘

📈 对应净值价位
- 现价：{净值}
- 加仓 1 触发价：{净值 × 0.95}（-5%）
- 加仓 2 触发价：{净值 × 0.90}（-10%）
- 加仓 3 触发价：{净值 × 0.85}（-15%）

📌 止损纪律：如净值跌破 {净值 × 0.80}，停止加仓，
   等待明确企稳信号后再评估。
📌 止盈纪律：浮盈 30% 后，先锁定 50% 利润。
```

#### 策略 C：网格交易

适用场景：指数基金、震荡市、有明确支撑/压力位

```
📋 网格交易计划 — {基金名称}（{代码}）
━━━━━━━━━━━━━━━━━━━━━━━
总资金：{XX} 万元
每格金额：¥{X}
当前净值：{净值}

┌──────┬──────────┬────────┬──────────┬────────┐
│ 网格 │  买入价  │ 买入金额 │  卖出价  │ 卖出金额 │
├──────┼──────────┼────────┼──────────┼────────┤
│  1   │ ¥{X}     │ ¥{Y}   │ ¥{X+Δ}   │ ¥{Y}   │
│  2   │ ¥{X-Δ}   │ ¥{Y}   │ ¥{X}     │ ¥{Y}   │
│  3   │ ¥{X-2Δ}  │ ¥{Y}   │ ¥{X-Δ}   │ ¥{Y}   │
│ ...  │ ...      │ ...    │ ...      │ ...    │
│  N   │ ¥{X}     │ ¥{Y}   │ ¥{X}     │ ¥{Y}   │
└──────┴──────────┴────────┴──────────┴────────┘

📐 网格间距：{Δ%}%（根据历史波动率调整）
📌 每笔交易手续费：约 {fee}%
📌 最低持仓：保留 1 格底仓不卖
📌 暂停条件：单边涨/跌超过 {N} 格，暂停网格
```

### 第四步：风险提示

```
⚠️ 执行纪律
━━━━━━━━━━━━━━━━━━━━━━━
1. 严格执行触发条件，不要临时改变计划
2. 每次操作记录在案，便于复盘
3. 每月回顾一次执行效果
4. 市场极端情况（涨跌停、熔断）暂停执行

⚠️ 免责声明：
本计划基于历史数据生成，不预测未来，
不构成投资建议。请根据自身风险承受能力决策。
```

## 注意事项

- 主动基金不适合网格交易（净值不连续、申赎费用高），仅建议用于 ETF/LOF
- 金字塔建仓的跌幅触发要合理（太密集容易过早打光子弹）
- DCA 在明显的估值极低区域（PE < 10% 分位）可以加倍定投
- 始终强调纪律执行，不要"计划很完美，执行随心所欲"
- 始终附带免责声明
