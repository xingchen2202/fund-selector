# 竞争分析命令

构建全面的中国市场竞争分析报告，涵盖产业链映射、竞争格局和政策环境。

## 工作流

### 第一步：解析输入

从输入中识别：
- 目标公司或行业
- 分析侧重点（如有指定）

如未提供，询问：
- "请输入要分析的公司名称或行业"

### 第二步：数据采集

使用 `cn-competitive-analysis` 技能：

1. **确定竞争格局**：
   - `get_company_info` — 目标公司主营和行业
   - `get_industry_stocks` — 同行业上市公司
   - `get_concept_list` — 相关概念板块

2. **采集财务数据**（对每家主要竞争对手）：
   - `get_income_statement` — 营收和盈利结构
   - `get_financial_indicators` — 经营效率指标
   - `get_realtime_quote` — 市值规模

### 第三步：分析

1. **产业链映射** — 上中下游关系
2. **波特五力** — 竞争强度评估
3. **竞争对手矩阵** — 多维度对标
4. **政策环境** — 产业/监管/贸易政策
5. **SWOT 分析** — 战略定位

### 第四步：输出

提供：
1. **分析报告**（10-15 页）— 完整竞争分析
2. **摘要** — 核心结论和投资/战略含义
