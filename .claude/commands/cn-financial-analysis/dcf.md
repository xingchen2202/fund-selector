# DCF 估值命令

构建 A 股公司 DCF（现金流折现）估值模型，输出 Excel 格式。

## 工作流

### 第一步：解析输入

从输入中识别：
- 目标公司（名称或 6 位代码）

如未提供，询问：
- "请输入要估值的公司名称或股票代码"

### 第二步：数据采集

使用 `cn-dcf-model` 技能：

1. **历史财务数据**（3-5 年）：
   - `get_income_statement` — 利润表
   - `get_balance_sheet` — 资产负债表
   - `get_cash_flow` — 现金流量表

2. **当前市场数据**：
   - `get_realtime_quote` — 当前股价、市值
   - `get_historical_price` — 历史股价（计算 Beta）
   - `get_company_info` — 基本信息

### 第三步：构建模型

1. **WACC 计算**（人民币口径）
2. **5-10 年自由现金流预测**
3. **终值计算**（永续增长法 + 退出倍数法）
4. **敏感性分析**（3 张 5×5 表格）
5. **情景分析**（悲观/基准/乐观）

### 第四步：输出

提供：
1. **Excel 模型** — 完整 DCF 工作簿（假设、三表预测、估值、敏感性）
2. **估值摘要** — 每股内在价值 vs 当前股价，上涨/下跌空间
