# 三表分析命令

按中国企业会计准则构建利润表、资产负债表、现金流量表分析与预测模型。

## 工作流

### 第一步：解析输入

从输入中识别：
- 目标公司（名称或 6 位代码）
- 分析范围（历史几年、预测几年）

如未提供，询问：
- "请输入公司名称或股票代码"
- 默认：历史 3 年 + 预测 3 年

### 第二步：数据采集

使用 `cn-3-statements` 技能：

1. **获取历史三表**：
   - `get_income_statement` — 利润表
   - `get_balance_sheet` — 资产负债表
   - `get_cash_flow` — 现金流量表

2. **获取辅助数据**：
   - `get_financial_indicators` — 财务指标（周转率、利润率）
   - `get_company_info` — 行业和业务信息

### 第三步：分析与建模

1. **整理历史数据** — 按中国会计准则标准科目
2. **计算关键比率** — 盈利、运营、偿债指标
3. **验证勾稽关系** — 三表交叉验证
4. **构建预测模型** — 基于驱动因子的前瞻预测

### 第四步：输出

提供：
1. **Excel 工作簿** — 历史三表 + 预测三表 + 比率分析 + 勾稽检查
2. **分析摘要** — 关键发现和预测假设说明
