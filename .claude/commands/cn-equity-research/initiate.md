# 首次覆盖命令

创建机构级首次覆盖报告（深度研报），30-50 页。

## 工作流

### 第一步：解析输入

识别目标公司。如未提供，询问。

### 第二步：使用 `cn-initiating-coverage` 技能

1. **公司概况**：通过 MCP 获取基本信息、财务数据
2. **行业分析**：行业空间、竞争格局、产业政策
3. **业务深度**：逐业务线分析
4. **财务分析**：盈利、资产质量、现金流、杜邦分解
5. **盈利预测**：3-5 年三表预测
6. **估值分析**：PE / PB-ROE / DCF / PEG 多方法
7. **风险提示**

### 第三步：输出

1. **DOCX 报告**（30-50 页）
2. **XLS 盈利预测模型**
3. **投资评级 + 目标价**
