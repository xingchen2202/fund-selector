# 并购分析命令

使用 `cn-merger-model` 技能，分析中国并购重组交易。

## 工作流

### 第一步：解析输入

识别：
- 收购方（上市公司）
- 标的方（拟收购资产/公司）
- 交易方式（如已知）

如信息不足，询问：
- "收购方是哪家上市公司？"
- "标的方基本情况？"
- "交易方式是什么？（发行股份/现金/混合）"

### 第二步：分析

1. **交易结构设计**：支付方式、发行价格
2. **标的估值**：DCF + 可比交易
3. **业绩承诺设计**：承诺利润 + 补偿机制
4. **增厚/摊薄分析**：EPS/ROE 影响
5. **监管审批判断**：审核路径 + 关注点

### 第三步：输出

1. **分析报告**（DOCX）
2. **并购模型**（XLS）
3. **交易方案建议摘要**
