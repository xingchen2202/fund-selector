# 路演材料命令

使用 `cn-pitch-deck` 技能，创建专业的投行路演 PPT。

## 工作流

### 第一步：解析输入

识别：
- 公司名称
- 路演场景（IPO / 并购推介 / 定增 / 债券发行）

如未提供，询问：
- "为哪家公司制作路演材料？"
- "什么场景？（IPO路演/并购推介/定增推介/债券发行推介）"

### 第二步：数据采集

通过 MCP 获取：
- 公司基本信息
- 财务数据（3 年）
- 行业对标数据
- 估值数据

### 第三步：制作

按场景模板生成 25-40 页 PPT：
- 投资亮点
- 公司介绍
- 行业分析
- 业务分析
- 财务分析
- 估值/定价

### 第四步：输出

PPTX 文件
