# IPO 分析命令

使用 `cn-ipo-analysis` 技能，分析 A 股 IPO 项目。

## 工作流

### 第一步：解析输入

识别：
- 拟上市公司（名称或描述）
- 目标板块（如已知）
- 关键财务数据（如已提供）

如信息不足，询问：
- "请提供公司名称和基本情况"
- "拟上市哪个板块？（主板/科创板/创业板/北交所）"

### 第二步：分析

1. **上市条件评估**：对照各板块条件
2. **板块选择建议**：基于公司特征推荐
3. **可比公司估值**：通过 MCP 获取同行数据
4. **发行定价区间**：基于可比估值
5. **审核关注点预判**

### 第三步：输出

1. **分析报告**（DOCX）
2. **估值模型**（XLS）
3. **核心结论**：推荐板块 + 估值区间
