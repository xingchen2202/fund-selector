# 债券发行分析命令

使用 `cn-bond-issuance` 技能，分析境内信用债发行方案。

## 工作流

### 第一步：解析输入

识别：
- 发行人名称
- 拟发行品种（公司债/企业债/短融/中票/ABS）
- 发行规模（如已知）

如信息不足，询问：
- "发行人是？"
- "拟发行什么品种？"

### 第二步：分析

1. **品种选择建议**：基于发行人资质和需求
2. **信用分析**：定量指标 + 定性评估
3. **发行定价**：基准利率 + 信用利差 + 可比定价
4. **偿债能力分析**：现金流覆盖测试
5. **发行条件评估**：是否满足监管要求

### 第三步：输出

1. **分析报告**（DOCX）
2. **信用分析模型**（XLS）
3. **定价建议**
